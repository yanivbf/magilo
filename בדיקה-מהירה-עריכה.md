## בדיקת שגיאות

### Console
1. פתח Developer Tools (F12)
2. לך ל-Console
3. בדוק אם יש שגיאות אדומות

### Network
1. פתח Developer Tools (F12)
2. לך ל-Network
3. ערוך משהו
4. בדוק אם הבקשה הצליחה (200 OK)

## תיקון בעיות נפוצות

### שגיאת 404 בהעלאת תמונות
**פתרון:** Restart לשרת
```bash
# עצור את השרט (Ctrl+C)
cd new-app
npm run dev
```

### שינויים לא נשמרים
**בדיקה:**
1. פתח Console (F12)
2. חפש שגיאות
3. בדוק אם Strapi רץ (port 1337)

### תמונות לא מוצגות
**בדיקה:**
1. בדוק אם הקובץ קיים ב-`strapi-backend/public/uploads/`
2. בדוק אם ה-URL נכון
3. נסה לפתוח את ה-URL ישירות בדפדפן
