#   拽专 砖专 "爪注"?

## 转 

### 1. 驻住 爪专转 祝
```javascript
// DynamicForm.svelte
let selectedStyle = $state('modern'); // 专专转 

// 砖专 "爪注":
selectedStyle = 'colorful';

// 砖转 驻住:
const data = {
    ...formData,
    designStyle: selectedStyle // 'colorful'
};
```

### 2. 爪专转 祝
```javascript
// /api/create-structured-page
const pageData = {
    designStyle: 'colorful', // 砖专 -Strapi
    // ... 砖专 转
};
```

### 3. 转爪转 祝
```svelte
<!-- view/[slug]/+page.svelte -->
<script>
const designStyle = data.page.designStyle || 'modern';
// designStyle = 'colorful'
</script>

<DynamicDesignWrapper {designStyle}>
    <!-- 转 祝 -->
</DynamicDesignWrapper>
```

### 4. 注 注爪
```javascript
// DynamicDesignWrapper.svelte
import { designSystems } from '$lib/designSystems.js';

const system = designSystems[designStyle]; // designSystems['colorful']

// 砖:
system.colors.primary    // #FF6B35 (转)
system.colors.secondary  // #FFD23F (爪)
system.colors.accent     // #EE4266 (专)
system.effects.shadows   // 'vibrant' (爪 爪注)
system.effects.animations // 'bouncy' (爪转 拽驻爪转)
```

##  专 转 砖?

### 爪注
- **驻转专**: 转-爪 拽 住-专
- **转专转**: 转 拽 住
- **专拽注**: 专 转-爪
- **拽砖专**: 转

### 驻拽
- **爪**: 爪注 (转/爪) 拽 驻专
- **爪转**: 拽驻爪转 (bouncy) 拽 拽转
- **转**: 注 转专 (4px)
- **专**: 转 爪

##  拽?
1. 驻转 转 祝 砖爪专转
2. 驻转 Developer Tools (F12)
3. 拽 转 -CSS Variables:
```css
--color-primary: #FF6B35;  /* 转 */
--color-secondary: #FFD23F; /* 爪 */
```

4. 拽 转 -data attribute:
```html
<div data-design-style="colorful">
```
