# ✅ תיקון רקע לבן והעלאת תמונה

## 🎨 בעיה 1: כל האתר לבן

### הבעיה
למרות שהדף מוגדר כ-`dark` ב-Strapi, כל האתר נראה לבן.

### הסיבה
היה CSS שדרס את הרקע:
```css
.modern-page {
    background: #ffffff; /* ❌ זה דרס את הרקע מה-DynamicDesignWrapper! */
}
```

### התיקון
הסרנו את הרקע הקשיח:
```css
.modern-page {
    min-height: 100vh;
    /* ✅ NO background - inherit from DynamicDesignWrapper! */
}
```

עכשיו הדף יורש את הרקע מה-`DynamicDesignWrapper` לפי העיצוב שנבחר:
- **dark**: `#0f172a` (כחול-שחור כהה)
- **modern**: `#ffffff` (לבן)
- **colorful**: `#ffffff` (לבן)
- וכו'...

## 📸 בעיה 2: לא ניתן להעלות תמונה לדף הראשי

### הבעיה
כפתור "העלה תמונה" לא עבד - התמונה לא נשמרה.

### הסיבה
הקוד שלח את הנתונים בפורמט לא נכון ל-API:
```javascript
// ❌ פורמט לא נכון
body: JSON.stringify({
    pageId: data.page.documentId,
    metadata: {
        headerImage: url
    }
})
```

ה-API מצפה לפורמט אחר:
```javascript
// ✅ פורמט נכון
body: JSON.stringify({
    pageId: data.page.documentId,
    field: 'metadata',
    value: {
        headerImage: url
    }
})
```

### התיקון
עדכנו את הקוד להשתמש בפורמט הנכון עם `field` ו-`value`.

---

## 🚀 מה לעשות עכשיו?

### 1. רענן את הדפדפן
```
Ctrl + Shift + R
```

### 2. בדוק את הרקע
- אם הדף מוגדר כ-**dark** - הרקע צריך להיות שחור (`#0f172a`)
- אם הדף מוגדר כ-**modern** - הרקע צריך להיות לבן (`#ffffff`)

### 3. נסה להעלות תמונה
1. לחץ על כפתור **"העלה תמונה"** בדף הצפייה
2. בחר תמונה מהמחשב
3. התמונה צריכה להופיע כרקע של ה-Hero section

## 📊 מה תוקן?

| בעיה | לפני | אחרי |
|------|------|------|
| רקע האתר | לבן תמיד | לפי העיצוב שנבחר |
| העלאת תמונה | לא עובד | עובד ✅ |
| רקע dark | לבן | שחור `#0f172a` |
| רקע modern | לבן | לבן `#ffffff` |

## 🔍 איך לבדוק?

### בדיקת רקע:
1. פתח את הדף בדפדפן
2. לחץ `F12` לפתיחת DevTools
3. בדוק את ה-CSS של `.design-wrapper`:
   ```css
   .design-wrapper {
       background-color: var(--color-bg) !important;
   }
   ```
4. בדוק את הערך של `--color-bg`:
   - dark: `#0f172a`
   - modern: `#ffffff`

### בדיקת העלאת תמונה:
1. לחץ על "העלה תמונה"
2. בחר תמונה
3. המערכת תציג: "⏳ מעלה תמונה..."
4. אחרי העלאה: "✅ תמונת הרקע עודכנה"
5. הדף יתרענן אוטומטית
6. התמונה תופיע כרקע של ה-Hero
