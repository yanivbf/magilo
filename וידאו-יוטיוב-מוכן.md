# 🎥 וידאו יוטיוב - מוכן!

## מה נוסף?

### תמיכה בסרטוני יוטיוב
הדף עכשיו מציג סרטוני יוטיוב אוטומטית אם מדביקים קישור!

## איך זה עובד?

### 1. הדבקת קישור יוטיוב
כשיוצרים דף חדש, אפשר להדביק קישור יוטיוב בשדה "וידאו":
- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/embed/VIDEO_ID`

### 2. המערכת מזהה אוטומטית
הקוד מחלץ את ה-VIDEO_ID מהקישור ויוצר iframe מוטמע.

### 3. הצגה בהירו
הסרטון מוצג בהירו (Hero Section) מתחת לכותרת והתיאור:

```
┌─────────────────────────────────────┐
│   Hero Section                      │
│   [כותרת ענקית]                     │
│   [תיאור]                           │
│   ┌───────────────────────────┐     │
│   │                           │     │
│   │   🎥 סרטון יוטיוב         │     │
│   │                           │     │
│   └───────────────────────────┘     │
│   [📞 התקשר עכשיו]                 │
└─────────────────────────────────────┘
```

## תכונות הוידאו

### 🎨 עיצוב מודרני
- מסגרת מעוגלת (border-radius: 24px)
- צל מרשים (box-shadow)
- יחס גובה-רוחב 16:9
- רוחב מקסימלי 800px
- אנימציית כניסה

### 📱 Responsive
- מתאים לכל גודל מסך
- שומר על יחס גובה-רוחב
- נראה מעולה במובייל

### ⚙️ הגדרות YouTube
- `rel=0` - לא מציג סרטונים קשורים
- `modestbranding=1` - מסתיר לוגו יוטיוב
- `allowfullscreen` - מאפשר מסך מלא
- תמיכה בכל תכונות יוטיוב (autoplay, gyroscope וכו')

## איך להשתמש?

### בעת יצירת דף:
1. פתח את יוצר הדפים
2. בחר סוג דף (חנות, אירוע וכו')
3. מלא את הפרטים
4. **הדבק קישור יוטיוב בשדה "וידאו"**
5. צור את הדף
6. הסרטון יופיע אוטומטית! 🎉

## קוד שנוסף

### קובץ: `new-app/src/routes/view/[slug]/+page.svelte`

#### 1. פונקציה לחילוץ ID
```javascript
function getYouTubeId(url) {
	if (!url) return null;
	const regExp = /^.*(youtu.be\/|v\/|u\/\w\/|embed\/|watch\?v=|&v=)([^#&?]*).*/;
	const match = url.match(regExp);
	return (match && match[2].length === 11) ? match[2] : null;
}
```

#### 2. חילוץ הקישור מ-metadata
```javascript
const videoUrl = data.page.metadata?.videoUrl || '';
const youtubeId = getYouTubeId(videoUrl);
```

#### 3. הצגת הוידאו
```svelte
{#if youtubeId}
	<div class="video-container">
		<iframe
			src="https://www.youtube.com/embed/{youtubeId}?rel=0&modestbranding=1"
			title="YouTube video"
			frameborder="0"
			allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
			allowfullscreen
		></iframe>
	</div>
{/if}
```

#### 4. עיצוב CSS
```css
.video-container {
	position: relative;
	width: 100%;
	max-width: 800px;
	margin: 2.5rem auto;
	padding-bottom: 56.25%; /* 16:9 */
	height: 0;
	overflow: hidden;
	border-radius: 24px;
	box-shadow: 0 20px 60px rgba(0, 0, 0, 0.4);
	animation: fadeInUp 1s ease-out 0.3s backwards;
}
```
