# 🔍 בדיקה מהירה - למה אין לי כפתורי עריכה?

## צעד 1: בדוק שאתה מחובר

פתח את הדף שלך והקלד בקונסול (F12):

```javascript
document.cookie
```

### מה אתה אמור לראות?
```
userId=user_1733410000000_abc123; ...
```

### אם אין `userId`:
❌ אתה לא מחובר!

**פתרון:**
1. לך ל-`http://localhost:5173/login`
2. התחבר
3. חזור לדף שלך

---

## צעד 2: בדוק מה הקונסול אומר

פתח קונסול (F12) ותראה:

### אם רואה:
```
👤 Is owner: true ✅
```
**הכל תקין!** אמור לראות כפתורי עריכה.

### אם רואה:
```
👤 Is owner: false ❌
```
**יש בעיה!** המערכת לא מזהה אותך כבעלים.

---

## צעד 3: תקן את הבעיה

### פתרון מהיר - צור דף חדש:
1. לך ל-Dashboard
2. לחץ "צור דף חדש"
3. הדף החדש יעבוד מושלם ✅

### פתרון לדפים קיימים - הרץ סקריפט:
```bash
node fix-ownership.js
```

זה יתקן את כל הדפים הישנים.

---

## מה אני אמור לראות אחרי התיקון?

### 1. בקונסול:
```
👤 Is owner: true ✅
```

### 2. בדף:
- ✅ כפתור "החלף רקע" בצד שמאל (עם אנימציה)
- ✅ סרגל עריכה עם 3 כפתורים למעלה
- ✅ סמלי מצלמה על תמונות הגלריה (עם אנימציה)

### 3. כשמעביר עכבר על תמונה:
- ✅ סמל מצלמה מופיע
- ✅ לחיצה פותחת בחירת תמונה חדשה

---

## עדיין לא עובד?

### בדוק את ה-userId שלך:
```javascript
// בקונסול:
const userId = document.cookie.split('; ').find(row => row.startsWith('userId='))?.split('=')[1];
console.log('Your userId:', userId);
```

### הרץ בדיקה מפורטת:
```bash
node debug-ownership.js YOUR-PAGE-SLUG
```

זה יראה לך בדיוק מה הבעיה.

---

## למה זה קורה?

דפים שנוצרו לפני התיקון האחרון לא נשמרו עם `createdByUserId`.

**דפים חדשים** שנוצרים עכשיו עובדים מושלם! ✅

**דפים ישנים** צריכים תיקון עם הסקריפט `fix-ownership.js`.
