### 1️⃣ **הוספת מוצרים לעגלה** 🛒
**דוגמאות למשפטי משתמש:**
- "אני רוצה 3 שוקולדים"
- "תוסיף לי 2 עוגות"
- "הוסף קפה"

**תגובת n8n צריכה להיות:**
```json
{
  "message": "✅ מעולה! הוספתי לך 3 שוקולדים לעגלה 🛒\nסה\"כ: ₪120",
  "action": "add_to_cart",
  "product": {
    "name": "שוקולד מריר",
    "price": 40,
    "image": "https://example.com/image.jpg"
  }
}
```

## 🔧 **איך לקונפג את n8n:**

### צעד 1: קבלת הבקשה מהמשתמש
ה-webhook שלך יקבל:
```json
{
  "message": "הודעת המשתמש",
  "pageContext": "URL של הדף"
}
```

### צעד 2: זיהוי הכוונה (Intent Recognition)
- השתמש ב-AI (OpenAI/Claude) כדי לזהות מה המשתמש רוצה
- בדוק מילות מפתח:
  - "להוסיף", "לקנות", "רוצה" → add_to_cart
  - "תראי", "המלצות", "מה יש" → scroll_to_products
  - "שאלות", "FAQ" → scroll_to_faq
  - "המלצות לקוחות", "ביקורות" → scroll_to_testimonials
  - "פרטי קשר", "איך ליצור קשר" → scroll_to_contact
  - "וואטסאפ", "WhatsApp" → open_whatsapp
  - "אישור הגעה", "אני מגיע" → fill_rsvp_form
  - "להירשם לקורס" → enroll_course

### צעד 3: החזרת תגובה עם action
החזר JSON עם:
- `message` - ההודעה שהבוט יציג למשתמש
- `action` - שם הפעולה לביצוע
- נתונים נוספים (product, formData, courseData וכו') לפי הצורך

---

## 💡 **טיפים חשובים:**

1. **תמיד תחזיר רק `message` טקסט** - אל תחזיר JSON שלם כהודעה!
2. **השתמש באימוג'ים** 🎉 - זה עושה את הבוט יותר ידידותי
3. **תן המלצות פרואקטיביות** - "האם תרצה לראות המלצות לקוחות?"
4. **תשמור על שיחה זורמת** - אל תהיה רובוטי מדי

## 🎯 **דוגמה מלאה לשיחה:**

```json
  "message": "כן! יש לנו שוקולד מריר מעולה במחיר 40₪. רוצה להוסיף לעגלה?"
```

```json
{

    "image": ""
  }
}
```
