## 🎯 איך להשתמש

### עריכת טקסט
1. **לחץ על הטקסט** - הוא יהפך לניתן לעריכה
2. **ערוך** - שנה את התוכן
3. **לחץ מחוץ לשדה** - שמירה אוטומטית!
4. **הודעה ירוקה** - "✅ נשמר בהצלחה"

### החלפת תמונה
1. **לחץ על התמונה** - יופיע overlay עם אייקון מצלמה
2. **בחר קובץ** - מתיקיית המחשב
3. **העלאה אוטומטית** - התמונה מוחלפת מיד
4. **הודעה ירוקה** - "✅ התמונה הועלתה בהצלחה"

### הוספת פריט
1. **לחץ על כפתור ➕** - בסוף הרשימה
2. **פריט חדש נוצר** - עם ערכי ברירת מחדל
3. **ערוך את הפרטים** - כמו כל טקסט אחר

### מחיקת פריט
1. **לחץ על כפתור 🗑️** - בפינה של הפריט
2. **אישור** - "האם למחוק?"
3. **הפריט נמחק** - מיד

## 🔧 טכנולוגיה

### רכיבים
- `EditableText.svelte` - עריכת טקסט
- `EditableImage.svelte` - החלפת תמונות

### Context API
```javascript
setContext('editMode', true);
setContext('pageId', documentId);
setContext('saveField', saveField);
```

### שמירה
כל שינוי נשמר מיד ל-Strapi:
```javascript
await saveField(`sections.${sectionIndex}.data.title`, value);
```
