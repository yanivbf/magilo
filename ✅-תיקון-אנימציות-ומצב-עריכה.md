## 📁 קבצים ששונו

### 1. `new-app/src/routes/view/[slug]/+page.svelte`
```javascript
// נוסף:
import { onMount } from 'svelte';
let isLoading = $state(true);

onMount(() => {
  setTimeout(() => {
    isLoading = false;
  }, 800);
});
```

**אנימציית טעינה:**
```html
{#if isLoading}
  <div class="autopage-loading">
    <div class="loading-logo">
      <div class="logo-box">A</div>
      <div class="loading-text">AutoPage</div>
    </div>
  </div>
{/if}
```

**CSS:**
- `.autopage-loading` - מסך מלא עם רקע סגול
- `.logo-box` - לוגו A מסתובב
- `@keyframes rotate` - סיבוב 360 מעלות
- `@keyframes fadeOut` - נעלם אחרי 0.8 שניות

**הסרת אנימציות:**
- הוסרה `animation: slideDown` מ-`.edit-toolbar-top`
- הוסרו כל ה-`animate-*` classes מהמקטעים
- הוסרה `animation: fadeInUp` מ-`.hero-content`
- הוסרה `animation: fadeInUp` מ-`.video-container`

**תיקון מצב עריכה:**
```css
/* רק במצב עריכה אמיתי */
:global(.edit-mode-active .editable:hover) {
  background: rgba(255, 255, 255, 0.1);
  outline: 2px dashed rgba(255, 255, 255, 0.3);
}
```
