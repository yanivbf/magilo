# ✅ עריכת תמונות בדפים חדשים - מוכן!

## איך זה עובד

### 1. יצירת דף חדש
כשאתה יוצר דף חדש:
```javascript
// page-creator/+page.svelte
sessionStorage.setItem('newlyCreatedPage', result.slug);
```
המערכת שומרת בזיכרון הדפדפן שזה דף חדש.

### 2. טעינת הדף
כשהדף נטען:
```javascript
// view/[slug]/+page.svelte
const newPageSlug = sessionStorage.getItem('newlyCreatedPage');
if (newPageSlug === data.page.slug) {
    isNewlyCreated = true;
    // מפעיל מצב עריכה
}
```

### 3. מצב עריכה
```javascript
const editMode = data.isOwner || isNewlyCreated;
setContext('editMode', editMode);
```
אם זה דף חדש → `editMode = true`

### 4. לחיצה על תמונה
כל תמונה עטופה ב-`EditableImage`:
- אם `editMode = true` → לחיצה פותחת בחירת תמונה
- אם `editMode = false` → רק תצוגה

## מה קורה עכשיו

1. **צור דף חדש** → לחץ "צור דף"
2. **הדף נוצר** → המערכת שומרת `newlyCreatedPage` בזיכרון
3. **הפניה לדף** → `/view/[slug]`
4. **המערכת בודקת** → "זה הדף שנוצר עכשיו?"
5. **מפעילה עריכה** → `editMode = true`
6. **לחץ על תמונה** → נפתח בחירת תמונה! 📸

## איפה זה עובד

### תמונות שניתן לערוך:
- ✅ תמונות בגלריה (`GallerySection`)
- ✅ תמונות מוצרים (`ProductsGallerySection`)
- ✅ תמונת רקע Hero (כפתור "החלף רקע")
- ✅ כל תמונה שעטופה ב-`EditableImage`

### איך זה נראה:
```svelte
{#if editMode}
    <EditableImage 
        src={image.url}
        onUpload={handleUpload}
    />
{:else}
    <img src={image.url} alt="" />
{/if}
```

## מתי זה מפסיק לעבוד

הזיכרון (`sessionStorage`) נמחק כש:
- סוגרים את הטאב
- רוענים את הדף (F5)
- עוברים לדף אחר וחוזרים

**זה בדיוק מה שרצית** - רק בדף שנוצר עכשיו, לא בדפים ישנים!

## בדיקה

1. **צור דף חדש**
2. **תראה בקונסול:**
   ```
   ✅ Marked page as newly created: ef0bfa04-page-...
   ✅ This is a newly created page - enabling edit mode
   🎨 Edit mode: true
   ```
3. **לחץ על תמונה** → אמור להיפתח בחירת קובץ!

## אם זה לא עובד

בדוק בקונסול (F12):
```javascript
// בדוק אם הדף מסומן כחדש
sessionStorage.getItem('newlyCreatedPage')

// בדוק את מצב העריכה
// אמור להיות: 🎨 Edit mode: true
```
