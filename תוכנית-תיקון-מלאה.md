# 📋 תוכנית תיקון מלאה - להפוך למערכת הישנה

## מה חסר במערכת החדשה?

### 1. ❌ סרגל ניווט למעלה (Navigation Bar)
**במערכת הישנה:**
```html
<header class="site-header">
    <div class="header-container">
        <div class="logo">מספרה 55</div>
        <nav class="nav-menu">
            <a href="#hero">בית</a>
            <a href="#about">אודות</a>
            <a href="#services">שירותים</a>
            <a href="#gallery">גלריה</a>
            <a href="#contact">צור קשר</a>
        </nav>
    </div>
</header>
```

**במערכת החדשה:** ❌ לא קיים

**פתרון:** צריך להוסיף ל-PageRenderer

---

### 2. ❌ כתב על התמונה בהירו
**במערכת הישנה:**
```html
<section class="hero" style="background-image: url(...)">
    <div class="hero-content">
        <h1>מספרה 55</h1>
        <p>תיאור...</p>
    </div>
</section>
```
הכתב היה **על התמונה** עם רקע שקוף/מטושטש

**במערכת החדשה:** הכתב מוצג אבל לא בדיוק כמו הישן

**פתרון:** לשנות את ה-hero section ב-PageRenderer

---

### 3. ❌ בוט שיוצר תוכן למקטעים
**במערכת הישנה:**
- הבוט היה יכול ליצור מקטעים חדשים
- להוסיף תוכן למקטעים קיימים
- לערוך טקסט

**במערכת החדשה:** הבוט רק עונה על שאלות

**פתרון:** צריך להוסיף פונקציונליות ליצירת תוכן

---

### 4. ❌ עריכה חיה (Live Editing)
**במערכת הישנה:**
```javascript
// לחיצה על "עריכה" הפכה את כל הטקסט ל-contentEditable
document.querySelectorAll('.editable').forEach(el => {
    el.contentEditable = true;
});

// העלאת תמונות ישירות
// שינוי צבעים
// עריכת טקסט
```

**במערכת החדשה:** עריכה רק דרך טפסים

**פתרון:** צריך להוסיף מצב עריכה חיה

---

## 🎯 סדר עדיפויות

### שלב 1: סרגל ניווט (הכי חשוב!)
1. ליצור קומפוננטה `NavigationBar.svelte`
2. להוסיף אותה ל-PageRenderer
3. לעצב אותה בדיוק כמו הישן

### שלב 2: כתב על התמונה
1. לשנות את ה-hero section
2. להוסיף overlay
3. למקם את הטקסט על התמונה

### שלב 3: בוט יוצר תוכן
1. להוסיף פונקציות ליצירת מקטעים
2. לחבר לבוט
3. לאפשר הוספת תוכן

### שלב 4: עריכה חיה
1. להוסיף מצב עריכה
2. contentEditable על כל הטקסטים
3. העלאת תמונות ישירה
4. שינוי צבעים
