## 🔧 ארכיטקטורה

### Context API
```javascript
// בדף הראשי
setContext('editMode', true);
setContext('pageId', documentId);
setContext('saveField', saveField);

// במקטעים
const editMode = getContext('editMode');
const pageId = getContext('pageId');
const saveField = getContext('saveField');
```

### שמירה
```javascript
// שמירת שדה פשוט
await saveField('title', 'כותרת חדשה');

// שמירת שדה במקטע
await saveField(`sections.${sectionIndex}.data.title`, 'כותרת');

// שמירת מערך
await saveField(`sections.${sectionIndex}.data.items`, items);
```

### העלאת תמונות
```javascript
const formData = new FormData();
formData.append('image', file);
formData.append('pageId', pageId);
formData.append('sectionId', sectionIndex);

const response = await fetch('/api/upload-section-image', {
  method: 'POST',
  body: formData
});
```
