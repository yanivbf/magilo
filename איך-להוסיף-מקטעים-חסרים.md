# 📋 איך להוסיף מקטעים חסרים לדף

## 🎯 הבעיה
סימנת מקטעים ביוצר הדפים אבל הם לא מופיעים בדף.

## ✅ הפתרון

### שלב 1: פתח את Strapi
1. גש ל-http://localhost:1337/admin
2. התחבר
3. לחץ על "Content Manager" בתפריט השמאלי
4. בחר "Page" מהרשימה
5. מצא את הדף שלך ולחץ עליו

### שלב 2: הוסף מקטעים חסרים

#### מקטע סרטון (Video):
```json
{
  "type": "video",
  "enabled": true,
  "data": {
    "title": "🎥 סרטון",
    "subtitle": "צפו בסרטון שלנו",
    "videoUrl": "https://www.youtube.com/watch?v=YOUR_VIDEO_ID",
    "description": "תיאור הסרטון"
  }
}
```

#### מקטע אודות (About):
```json
{
  "type": "about",
  "enabled": true,
  "data": {
    "title": "ℹ️ אודותינו",
    "content": "ספר על העסק שלך...",
    "image": "",
    "features": [
      {
        "icon": "✨",
        "title": "תכונה 1",
        "text": "תיאור התכונה"
      }
    ]
  }
}
```

#### מקטע צוות (Team):
```json
{
  "type": "team",

    "title": "👥 הצוות שלנו",
    "subtitle": "הכירו את האנשים שלנו",
    "members": [

        "name": "שם חבר צוות",
        "role": "תפקיד",

```

#### מקטע שירותים (Services):
```json

      {

        "description": "תיאור השירות",

```

#### מקטע מחירון (Pricing):
```json

  "enabled": true,

```

### שלב 3: שמור את השינויים
1. לחץ על "Save" בפינה הימנית העליונה
2. רענן את הדף בדפדפן
3. המקטעים אמורים להופיע!

## 🔍 איך לבדוק מה חסר?

### בדוק ב-Console:
1. פתח את הדף
2. לחץ F12 (Developer Tools)
3. לחץ על "Console"
4. חפש את השורה: `📋 Page has X sections`
5. זה יראה לך כמה מקטעים יש בדף

### רשימת כל המקטעים הזמינים:
1. ✅ **gallery** - גלריית תמונות
2. ✅ **products** - מוצרים
3. ✅ **services** - שירותים
4. ✅ **pricing** - מחירון
5. ✅ **about** - אודות
6. ✅ **team** - צוות
7. ✅ **testimonials** - ממליצים
8. ✅ **faq** - שאלות ותשובות
9. ✅ **video** - סרטון
10. ✅ **contact** - צור קשר

## 💡 טיפים

### סדר המקטעים:
המקטעים מוצגים לפי הסדר שהם מופיעים במערך `sections` ב-Strapi.
אפשר לשנות את הסדר על ידי גרירה ב-Strapi.

### הפעלה/כיבוי מקטעים:
כל מקטע יש לו שדה `enabled`.
- `true` - המקטע מוצג
- `false` - המקטע מוסתר

### עריכה מהירה:
אחרי שהמקטע מוצג בדף, אפשר לערוך אותו ישירות בדף!
פשוט לחץ על הטקסט או התמונה וערוך.

## 🚨 בעיות נפוצות
