═══════════════════════════════════════════════════════════════
  1️⃣ תיקון: יומן תפוסות בדף התורים
═══════════════════════════════════════════════════════════════

❌ **הבעיה**:
   בדפי תורים לא היה לוח זמנים שמראה אילו שעות תפוסות

✅ **התיקון**:
   עכשיו כשהבוט יוצר דף תורים, הוא **חייב** לכלול:
   
   📅 **סקשן יומן/לוח זמנים (id="schedule-view")**:
   
   ✓ תצוגה יומית/שבועית של שעות
   ✓ כל שעה מראה:
      - 🟢 פנוי (ירוק)
      - 🔴 תפוס (אדום/אפור)
   ✓ ניווט (היום, מחר, +2 ימים...)
   ✓ לחיצה על שעה פנויה → ממלא אוטומטית את הטופס
   
   🔧 **JavaScript אוטומטי**:
   - מושך תורים קיימים מ: /api/appointments/[userId]/[pageId]
   - מציג אותם ביומן
   - מעדכן כשתור חדש נקבע
   - לחיצה על שעה פנויה ממלאת את התאריך והשעה בטופס


═══════════════════════════════════════════════════════════════
  2️⃣ תיקון: בועת הבוט נפתחת בלחיצה
═══════════════════════════════════════════════════════════════

❌ **הבעיה**:
   בועת הבוט לא נפתחת בלחיצה / לא אחידה בדפים

✅ **התיקון**:
   עדכנתי את ההנחיות שהבוט **חייב** לעבוד ככה:
   
   🤖 **בועת הבוט**:
   ✓ מיקום קבוע: פינה ימנית תחתונה
   ✓ צבע סגול (#8B5CF6)
   ✓ סטטוס ברירת מחדל: **סגור**
   
   🖱️ **לחיצה על הבועה**:
   → פותח חלון צ'אט (300x400px)
   → חלון מופיע ליד הבועה
   → אנימציה חלקה (slide-in)
   
   💬 **חלון הצ'אט**:
   ✓ שדה הקלדה + כפתור שליחה
   ✓ עובד גם עם Enter
   ✓ היסטוריית שיחה עם גלילה
   ✓ כפתור X לסגירה
   ✓ לחיצה שוב על הבועה → סוגר
   
   📋 **ניהול היסטוריה**:
   ✓ כל דף מקבל מפתח ייחודי
   ✓ לא מערבב שיחות בין דפים


═══════════════════════════════════════════════════════════════
  🧪 איך לבדוק?
═══════════════════════════════════════════════════════════════

1️⃣ **רענן את הדפדפן**: Ctrl+Shift+R

2️⃣ **צור דף תורים חדש**:
   → http://localhost:3002/page-creator/page-creator.html
   → בחר "קביעת תורים"
   → מלא את הטופס
   → צור דף

3️⃣ **בדוק בדף שנוצר**:
   ✅ צריך להיות **יומן תפוסות** עם שעות
   ✅ בועת בוט בפינה ימנית
   ✅ לחיצה על הבועה → פותח חלון צ'אט
   ✅ לחיצה שוב → סוגר
