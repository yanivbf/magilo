# 🔧 תיקון: מעצב שיער + Synonyms

## 🐛 הבעיה שהייתה:

כשמחפשים:
- ❌ **"מעצב שיער"** → לא מצא כלום
- ❌ **"יניב חן מעצב שיער"** → לא מצא
- ✅ **"מספרה"** → מצא (רק המונח הזה עבד)

---

## ✅ מה תיקנתי?

### 1. הוספתי `categoryMapping` ב-`marketplace.html`
```javascript
'מספרה': ['מספרה', 'תספורת', 'מעצב שיער', 'עיצוב שיער', 'ספר', 'barber', 'להסתפר', 'הסתפרות', 'מסתפר']
```

### 2. הוספתי `stavCategoryMapping` ב-`index.html`
```javascript
'מספרה': ['מספרה', 'תספורת', 'מעצב שיער', 'עיצוב שיער', 'ספר', 'barber', 'להסתפר', 'הסתפרות', 'מסתפר']
```

### 3. הוספתי פונקציה `expandStavSearchTerms()` ב-`index.html`
הפונקציה הזו לוקחת מונח חיפוש (למשל "מעצב שיער") ומרחיבה אותו לכל ה-synonyms (מספרה, תספורת, וכו')

### 4. עדכנתי את `searchLivePagesStav()` ב-`index.html`
עכשיו הפונקציה מחפשת בכל ה-synonyms ולא רק במונח המדויק:
```javascript
const found = expandedTerms.some(term => htmlLower.includes(term));
```
