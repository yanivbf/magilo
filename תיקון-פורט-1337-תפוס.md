# 🔧 תיקון: הפורט 1337 תפוס

## הבעיה
```
error: The port 1337 is already used by another application.
```

Strapi לא יכול להתחיל כי הפורט 1337 כבר בשימוש.

## הפתרון

### אופציה 1: סגור את התהליך הישן (מומלץ)
```cmd
netstat -ano | findstr :1337
```

זה יראה לך את ה-PID של התהליך. אחר כך:
```cmd
taskkill /PID <מספר_ה_PID> /F
```

### אופציה 2: סגור הכל ופתח מחדש
1. סגור את כל הטרמינלים
2. פתח Task Manager (Ctrl+Shift+Esc)
3. חפש "node" ב-Processes
4. סגור את כל תהליכי Node.js
5. פתח טרמינל חדש והרץ:

```cmd
cd strapi-backend
npm run develop
```

### אופציה 3: שנה את הפורט (זמני)
ערוך את `strapi-backend/.env`:
```
PORT=1338
```

אבל אז צריך לשנות גם ב-`new-app/.env`:
```
STRAPI_URL=http://localhost:1338
```

## מה לעשות עכשיו?

1. **סגור את כל הטרמינלים**
2. **פתח טרמינל חדש**
3. **הרץ:**
```cmd
cd strapi-backend
npm run develop
```

4. **חכה עד שתראה:**
```
✔ Strapi is running on http://localhost:1337
```

5. **אז פתח טרמינל נוסף והרץ:**
```cmd
cd new-app
npm run dev
```
