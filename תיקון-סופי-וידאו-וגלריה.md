```javascript
  label: '📘 קישור לפייסבוק (אופציונלי)',

  label: '📸 קישור לאינסטגרם (אופציונלי)',
  type: 'text'

```

### 2. תיקון פרסור הגלריה ✅

**קובץ:** `new-app/src/routes/api/create-page-with-template/+server.js`

**לפני:**
```javascript
gallery: pageData.gallery || []  // ❌ לא עובד אם זה string
```

**אחרי:**
```javascript
// Parse gallery from textarea
let galleryImages = [];
if (pageData.gallery) {
  if (Array.isArray(pageData.gallery)) {
    galleryImages = pageData.gallery;
  } else if (typeof pageData.gallery === 'string') {
    // Split by newlines and filter empty lines
    galleryImages = pageData.gallery
      .split('\n')
      .map(url => url.trim())
      .filter(url => url.length > 0 && url.startsWith('http'));
  }
}
```
