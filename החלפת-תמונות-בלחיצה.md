# 📸 החלפת תמונות בלחיצה

## מה נוסף

### ✅ תמונות מוצרים ניתנות להחלפה
כשמרחפים מעל תמונת מוצר, מופיע overlay סגול עם אייקון מצלמה ו"החלף תמונה"

## איך זה עובד

### 1. רחף מעל תמונת מוצר
```
תראה overlay סגול עם אייקון מצלמה 📸
```

### 2. לחץ על התמונה
```
נפתח חלון בחירת קובץ
```

### 3. בחר תמונה חדשה
```
התמונה מועלית אוטומטית
```

### 4. התמונה מתעדכנת
```
הודעה: "✅ התמונה הוחלפה בהצלחה"
```

## מה קורה מאחורי הקלעים

1. **לחיצה על תמונה** → פותח file input
2. **בחירת קובץ** → מעלה ל-Strapi
3. **Strapi שומר** → מחזיר URL חדש
4. **עדכון מוצר** → שומר את ה-URL החדש
5. **עדכון תצוגה** → התמונה מתעדכנת מיד

## קבצים שנוצרו

```
new-app/src/routes/api/upload-product-image/+server.js

└── ✅ מחזיר את ה-URL החדש

├── ✅ נוסף פונקציית handleImageUpload()
├── ✅ נוסף פונקציית showNotification()
└── ✅ נוסף סטיילים ל-overlay והתראות
```

## עיצוב ה-Overlay

### כשלא מרחפים:
- Overlay שקוף (opacity: 0)
- לא נראה

### כשמרחפים:
- Overlay סגול (rgba(102, 126, 234, 0.9))
- אייקון מצלמה לבן
- טקסט "החלף תמונה"
- אנימציה חלקה

### אחרי העלאה:
- הודעה ירוקה למטה
- "✅ התמונה הוחלפה בהצלחה"
- נעלמת אחרי 2 שניות

## איך לבדוק

1. פתח דף עם מוצרים
2. רחף מעל תמונת מוצר
3. תראה overlay סגול עם "החלף תמונה"
4. לחץ על התמונה
5. בחר תמונה חדשה מהמחשב
6. המתן להעלאה
7. תראה הודעה: "✅ התמונה הוחלפה בהצלחה"
8. התמונה מתעדכנת מיד!
