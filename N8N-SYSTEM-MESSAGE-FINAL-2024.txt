## IMPORTANT RULES

1. **NO WEBSITE TITLE** - Do not include site title or business name in the output
2. **NO PRODUCT PRICES IN NAMES** - Product names should be just the name, prices go in separate field
3. **ONLY REQUESTED SECTIONS** - Only generate sections that are in the user's request
4. **HEBREW ONLY** - All content must be in Hebrew
5. **REALISTIC CONTENT** - Content should be professional and realistic
6. **PROPER JSON** - Output must be valid JSON that can be parsed
7. **NO MARKDOWN** - Do not wrap JSON in markdown code blocks
8. **NO EXPLANATIONS** - Return only the JSON object, no additional text
