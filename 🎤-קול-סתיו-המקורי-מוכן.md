- עברית רובוטית מהירה
- קול נשי רובוטי עם אפקט מיוחד
- **בדיוק כמו סתיו הישנה!**

### 2. איפה הקול פועל?

**בשני הצ'אטים:**
1. **ChatBotBubble** - הצ'אט הכללי בדפים
2. **StavBot** - הצ'אט החכם במרקטפלייס

כל הודעה של הבוט מקבלת כפתור רמקול 🔊

## 📋 הגדרת API Key

### צריך להוסיף ל-.env:

```env
GOOGLE_TTS_API_KEY=your_google_api_key_here
```

### איך מקבלים API Key?

1. לך ל-[Google Cloud Console](https://console.cloud.google.com/)
2. צור פרויקט חדש (או בחר קיים)
3. הפעל את ה-API:
   - לך ל-"APIs & Services" > "Library"
   - חפש "Cloud Text-to-Speech API"
   - לחץ "Enable"
4. צור API Key:
   - לך ל-"APIs & Services" > "Credentials"
   - לחץ "Create Credentials" > "API Key"
   - העתק את ה-Key

### הוסף ל-.env:
```env
GOOGLE_TTS_API_KEY=AIzaSy...your_key_here
```
