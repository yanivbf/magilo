# מיפוי מלא של שדות טופס - נותן שירות עם יומן

## 📋 שדות בסיסיים
- `mainName` / `businessName` - שם העסק
- `description` / `tagline` - תיאור/סלוגן
- `phone` - טלפון
- `email` - אימייל
- `address` - כתובת
- `city` - עיר

## 🎨 עיצוב וסגנון
- `selectedStyle` / `designStyle` - סגנון עיצוב (Minimalist Pro, Midnight, Claymorphism, Retro Magazine, Aurora UI, Bento Grid, Modern, Minimalist, Vintage, Luxury, Gothic, Neon)
- `colorScheme` - ערכת צבעים
- `fontChoice` - בחירת פונט

## 🌐 רשתות חברתיות
- `facebook` - קישור לפייסבוק
- `instagram` - קישור לאינסטגרם
- `linkedin` - קישור ללינקדאין
- `twitter` / `x` - קישור לטוויטר/X
- `tiktok` - קישור לטיקטוק
- `youtube` - קישור ליוטיוב
- `whatsapp` - מספר וואטסאפ

## 📅 הגדרות יומן ותורים
- `enableAppointmentBooking` - הפעלת מערכת תורים
- `workStartTime` - שעת התחלת עבודה
- `workEndTime` - שעת סיום עבודה
- `workDays[]` - ימי עבודה (sunday, monday, tuesday, wednesday, thursday, friday, saturday)

### שעות מיוחדות לימים ספציפיים
- `specialDay[]` - יום עם שעות מיוחדות
- `specialStartTime[]` - שעת התחלה מיוחדת
- `specialEndTime[]` - שעת סיום מיוחדת

### הפסקות
- `breakStart[]` - התחלת הפסקה
- `breakEnd[]` - סיום הפסקה
- `breakDescription[]` - תיאור הפסקה

### שירותים
- `serviceName[]` - שם השירות
- `serviceDuration[]` - משך בדקות
- `servicePrice[]` - מחיר
- `serviceDescription[]` - תיאור השירות (אופציונלי)

## 🖼️ גלריה
- `galleryImages[]` - URLים של תמונות לגלריה
- `galleryTitles[]` - כותרות לתמונות
- `galleryDescriptions[]` - תיאורים לתמונות

## 💬 לקוחות מליצים (Testimonials)
- `testimonialName[]` - שם הלקוח
- `testimonialText[]` - טקסט ההמלצה
- `testimonialRating[]` - דירוג (1-5 כוכבים)
- `testimonialImage[]` - תמונת הלקוח (אופציונלי)

## 📝 תוכן עשיר (יבוא מ-N8N)
- `heroText` - טקסט Hero
- `aboutText` - טקסט אודות
- `faqItems[]` - שאלות ותשובות נפוצות

## 🎯 SEO ומטא
- `metaTitle` - כותרת SEO
- `metaDescription` - תיאור SEO
- `keywords` - מילות מפתח

## 🌍 שפה
- `outputLanguage` - שפת הפלט (he, en, ar, ja, es, fr, de, ru, zh)

3. גלריה (אם יש)
4. Testimonials (אם יש)
5. רשתות חברתיות
6. עיצוב לפי הסגנון שנבחר

### מ-N8N (רק מלל):

4. תיאורים שיווקיים
