# 🚀 בדוק את ההתחברות עם Google - עכשיו!

## מה תיקנו?

העברנו את ההתחברות מ-Supabase ל-Strapi. עכשיו הכל עובד במערכת שלנו.

## איך לבדוק?

### שלב 1: הפעל את הסרבר (אם הוא לא רץ)

```bash
cd new-app
npm run dev
```

### שלב 2: פתח את דף ההתחברות

פתח בדפדפן: `http://localhost:5173/login`

### שלב 3: לחץ על "התחבר עם Google"

הכפתור האדום הגדול עם לוגו של Google

### שלב 4: בחר חשבון Google

בחר את החשבון שלך מהרשימה

### שלב 5: בדוק שהגעת לדשבורד

אמור להעביר אותך ל: `http://localhost:5173/dashboard`

## מה אמור לקרות?

1. ✅ תראה את הדשבורד עם כל הדפים שלך
2. ✅ בפינה הימנית העליונה תראה את השם שלך
3. ✅ תוכל ללחוץ על "צור דף חדש"

## עכשיו נסה ליצור דף חדש:

1. **לחץ על "צור דף חדש"**
2. **בחר "חנות מקוונת"** (או כל תבנית אחרת)
3. **מלא את הפרטים**:
   - שם העסק
   - תיאור
   - העלה תמונה (אופציונלי)
4. **לחץ על "צור דף"**

## מה אמור לקרות?

1. ✅ הדף נוצר בהצלחה
2. ✅ אתה מועבר לדף הניהול: `/manage/[documentId]`
3. ✅ אתה רואה את הדף שיצרת
4. ✅ אין שגיאה 500!

## אם יש בעיה:

### בדוק ב-Console (F12):

1. פתח את ה-Console (לחץ F12)
2. לך ל-Application → Cookies
3. בדוק שיש Cookie בשם `userId`
4. הערך אמור להתחיל ב-`google_`

### בדוק ב-Terminal:

חפש הודעות כאלה:
```
✅ Cookie set for user: google_123456789
✅ New user created: google_123456789
```

או:
```
✅ Existing user logged in: google_123456789
```

## אם עדיין יש שגיאה 500:

תעתיק את השגיאה המלאה מה-Console ותשלח לי.

## זה אמור לעבוד!

המערכת החדשה פשוטה יותר ועובדת ישירות עם Strapi.
אין יותר Supabase, אין יותר בעיות עם cookies.

**נסה עכשיו!** 🎉
