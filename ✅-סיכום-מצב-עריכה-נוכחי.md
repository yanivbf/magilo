## מה לא עובד ❌

### עריכת טקסט במקטעים
- מקטע "אודות" (AboutSection)
- מקטע "מוצרים" (ProductsGallerySection)
- מקטעים אחרים

**השגיאה:**
```
Failed to update page
Invalid key documentId
```

**הסיבה:**
המקטעים משתמשים בפונקציות שמירה נפרדות (`saveProductField`, `saveAboutField`) שעדיין לא מעודכנות להשתמש ב-`documentId`.

---

## למה הכותרת עובדת והמקטעים לא?

### הכותרת משתמשת ב:
```javascript
// +page.svelte - saveField (עובד!)
const pageId = data.page.documentId || data.page.id;
```

### המקטעים משתמשים ב:
```javascript
// ProductsGallerySection.svelte - saveProductField (לא עובד!)
// עדיין משתמש ב-ID ישן או בלוגיקה שונה
```

---

## מה צריך לתקן?

1. **למצוא את כל הפונקציות שמירה במקטעים**
   - `saveProductField` ב-ProductsGallerySection
   - `saveAboutField` ב-AboutSection
   - פונקציות דומות במקטעים אחרים

2. **לוודא שכולן משתמשות ב-documentId**
   ```javascript
   const pageId = data.page.documentId || data.page.id;
   ```

3. **לוודא שה-API תומך ב-documentId**
   - ✅ כבר תוקן ב-`/api/update-page`
