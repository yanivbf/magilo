# 🧪 בדיקת עריכה מהירה

## תרחיש בדיקה

### 1. הכנה
```bash
# ודא שהשרתים רצים
cd new-app
npm run dev

# בטאב נפרד
cd strapi-backend
npm run develop
```

### 2. צור דף חדש
1. היכנס ל-http://localhost:5173
2. התחבר למערכת
3. לחץ "צור דף חדש"
4. בחר "חנות מקוונת"
5. מלא פרטים:
   - שם: "חנות הבדיקה שלי"
   - תיאור: "זו חנות לבדיקה"
   - עיר: "תל אביב"
6. לחץ "צור דף"

### 3. בדוק עריכה מהירה
1. היכנס לדף שיצרת
2. **בדוק שאתה רואה** את כפתור "✏️ עריכה מהירה" (שמאל למטה)
3. לחץ על הכפתור

### 4. בדוק עריכת טקסט
1. **hover על הכותרת** - אמור לראות outline ירוק בהיר
2. **לחץ על הכותרת** - אמור לקבל outline ירוק מקווקו
3. **ערוך את הכותרת** - שנה ל-"חנות הבדיקה המעודכנת"
4. **לחץ מחוץ לכותרת** - אמור לראות "שומר..."
5. **רענן את הדף** - הכותרת אמורה להישאר מעודכנת

### 5. בדוק עריכת תיאור
1. **לחץ על התיאור** (מתחת לכותרת)
2. **ערוך** - שנה ל-"תיאור מעודכן לבדיקה"
3. **לחץ מחוץ לתיאור**
4. **רענן** - התיאור אמור להישאר מעודכן

### 6. בדוק החלפת תמונה
1. **hover על תמונה** - אמור לראות outline סגול בהיר
2. **לחץ על תמונה** - אמור לקבל outline סגול מקווקו
3. **בחר תמונה חדשה** מהמחשב
4. **המתן** - התמונה אמורה להתעדכן אוטומטית
5. **רענן** - התמונה החדשה אמורה להישאר

### 7. סיים עריכה
1. לחץ "✅ סיים עריכה"
2. כל ה-outlines אמורים להיעלם
3. הדף אמור לחזור למצב רגיל

## 🐛 בעיות אפשריות

### הכפתור לא מופיע
- ודא שאתה מחובר
- ודא שאתה בעל הדף (לא מבקר)
- בדוק ב-console אם יש שגיאות

### השמירה לא עובדת
- בדוק ב-Network tab אם הבקשה נשלחת
- ודא ש-Strapi רץ
- בדוק את ה-console לשגיאות

### התמונה לא מתעדכנת
- ודא שהתמונה לא גדולה מדי (מקס 5MB)
- בדוק שה-API של upload-image עובד
- בדוק ב-Network tab את התגובה
