# ✅ מקטע שאלות ותשובות מוכן עם כותרת משנה רלוונטית

## 🎯 מה תוקן?

### 1. **כותרת משנה רלוונטית לעסק**
הבוט עכשיו יוצר כותרת משנה שמתאימה לסוג העסק:
- **לק גל** → "כל מה שרצית לדעת על לק גל ומניקור"
- **מספרה** → "שאלות נפוצות על תספורות ועיצוב שיער"
- **רופא שיניים** → "תשובות לשאלות על טיפולי שיניים"

### 2. **עדכון הנחיות הבוט**
קובץ: `📋-העתק-לN8N-עכשיו.txt`

הוספתי ל-JSON שהבוט מחזיר:
```json
{
  "faq": [...],
  "faqSubtitle": "כותרת משנה רלוונטית לעסק",
  "gallery": [...],
  ...
}
```

### 3. **עדכון קוד השרת**
קובץ: `new-app/src/routes/api/create-structured-page/+server.js`

השרת עכשיו:
- קורא את `faqSubtitle` מהבוט
- שומר אותו במקטע FAQ
- משתמש בכותרת משנה ברירת מחדל אם הבוט לא החזיר

```javascript
// Get contextual subtitle from AI or use default
const faqSubtitle = aiGeneratedContent?.faqSubtitle || 'תשובות לשאלות הנפוצות ביותר';

const faqSectionData = {
  type: 'faq',
  enabled: true,
  order: getSectionOrder('faq'),
  data: {
    title: 'שאלות ותשובות',
    subtitle: faqSubtitle,  // ✅ כותרת משנה רלוונטית
    items: faqItems
  },
  page: pageId
};
```

## 📋 מה צריך לעשות עכשיו?

### **העתק את ההנחיות המעודכנות ל-N8N:**

1. **פתח את N8N**: https://n8n-service-how4.onrender.com
2. **פתח את ה-workflow** של יצירת תוכן
3. **לחץ על ה-AI node** (OpenAI/Claude)
4. **העתק את התוכן** מהקובץ `📋-העתק-לN8N-עכשיו.txt`
5. **הדבק ב-System Message** (Options → System Message)
6. **שמור** (Save)

## ✅ מה כבר עובד?

1. ✅ **מקטע FAQ קיים** - הקומפוננטה מוכנה
2. ✅ **תמיכה בכותרת משנה** - הקומפוננטה כבר תומכת ב-subtitle
3. ✅ **יצירת מקטע** - השרת יוצר את המקטע כשמסמנים checkbox
4. ✅ **תצוגה בדף** - המקטע מוצג בדף הצפייה
5. ✅ **עריכה** - אפשר לערוך את הכותרת המשנה בדף

## 🔍 איך לבדוק?

1. **צור דף חדש** בעמוד יוצר הדפים
2. **סמן את checkbox "שאלות ותשובות"**
3. **מלא שם עסק** (למשל: "לק גל של שרה")
4. **לחץ "צור דף"**
5. **בדוק שהמקטע מופיע** עם כותרת משנה רלוונטית

## 📝 הערות חשובות

- הבוט יוצר כותרת משנה **רלוונטית לסוג העסק**
- אם הבוט לא מחזיר כותרת משנה, יש ברירת מחדל: "תשובות לשאלות הנפוצות ביותר"
- הכותרת המשנה **ניתנת לעריכה** בדף הצפייה (אם אתה הבעלים)
