# 🚀 האתר פתוח ורץ!

## ✅ השרתים רצים

1. **SvelteKit (Frontend)** - http://localhost:5173
2. **Strapi (Backend)** - http://localhost:1337

## 🎯 איך להשתמש?

### 1. פתח את האתר
```
http://localhost:5173
```

### 2. התחבר
- לחץ על "התחבר"
- התחבר עם Google

### 3. צור דף חדש
- לחץ על "צור דף חדש"
- בחר תבנית (חנות, שירות, אירוע, וכו')
- מלא פרטים
- לחץ "צור דף"

### 4. צפה בדף
- הדף ייפתח אוטומטית
- **תראה את האתר המלא ללא סרגל עריכה!**
- בדיוק כמו האתרים הישנים

## 🔧 פקודות שימושיות

### להפעיל את האתר (אם לא רץ):
```bash
cd new-app
npm run dev
```

### להפעיל את Strapi (אם לא רץ):
```bash
cd strapi-backend
npm run develop
```

### לעצור הכל:
```bash
Ctrl + C
```
