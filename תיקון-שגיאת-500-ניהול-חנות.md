# ✅ תיקון שגיאת 500 בדף ניהול חנות


## הסיבה
הדף נוצר עם `pageType: "store"` אבל ללא שדה `template`.
הקומפוננט של דף הניהול בדק את שדה `template` כדי להחליט איזה ממשק להציג.

## הפתרון
הוספתי fallback בקוד שבודק אם `template` חסר, והוא משתמש ב-`pageType` במקום:

```javascript
// Ensure template field exists (fallback to pageType if missing)
if (!page.template && page.pageType) {
	page.template = page.pageType;
	console.log('⚠️ Template field missing, using pageType:', page.pageType);
}
```
