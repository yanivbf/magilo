### 2. הוספת סגנונות ספציפיים לכל עיצוב
כל עיצוב מקבל עכשיו:
- צבעי רקע ייחודיים
- סגנונות Hero מיוחדים
- צבעי טקסט קריאים
- אפקטים ייחודיים (נאון זוהר, רטרו פסים וכו')

### 3. שימוש ב-!important
כדי לוודא שהסגנונות באמת חלים, הוספתי `!important` למקומות קריטיים.

### 4. סגנונות למקטעים זוגיים
מקטעים זוגיים מקבלים רקע חלופי (`backgroundAlt`) ליצירת ניגודיות.

## 🎯 איך זה עובד עכשיו

### בעת יצירת דף:
1. בוחרים עיצוב (מודרני, צבעוני, אלגנטי וכו')
2. העיצוב נשמר בשדה `designStyle` ב-Strapi
3. הדף נטען עם ה-`DynamicDesignWrapper`
4. כל המקטעים מקבלים את הצבעים והסגנונות של העיצוב

### מה קורה בקוד:
```javascript
// 1. העיצוב נטען מהדאטה
const designStyle = data.page.designStyle || 'modern';

// 2. DynamicDesignWrapper עוטף את כל הדף
<DynamicDesignWrapper {designStyle}>
	<!-- כל התוכן כאן -->
</DynamicDesignWrapper>

// 3. CSS Variables מוחלים
--color-primary: #f97316;
--color-bg: #ffffff;
--color-text: #1f2937;
// וכו'...

// 4. סגנונות ספציפיים לעיצוב
.design-colorful .hero-section {
	background: gradient(...);
	color: white;
}
```
