# 📸 איך לשתף את ה-Console - מדריך מהיר

## 🎯 מה אני צריך

אני צריך לראות מה קורה ב-Console כשאתה לוחץ על המיקרופון.

## 📋 3 שלבים פשוטים

### שלב 1️⃣: פתח Console
```
לחץ F12
(או Ctrl+Shift+I)
```

תראה חלון שנפתח בתחתית/צד של הדפדפן.

### שלב 2️⃣: לחץ על "Console"
```
למעלה בחלון שנפתח, תראה לשוניות:
Elements | Console | Sources | ...

לחץ על "Console"
```

### שלב 3️⃣: נקה ולחץ על המיקרופון
```
1. לחץ על האייקון 🚫 (Clear console) - זה ינקה הכל
2. לחץ על כפתור המיקרופון 🎤 בצ'אט של סתיו
3. תראה הרבה הודעות ב-Console
```

## 📸 עכשיו שתף איתי

### אופציה 1: צילום מסך
```
Win + Shift + S (Windows)
או
Cmd + Shift + 4 (Mac)

צלם את כל ה-Console ושלח לי
```

### אופציה 2: העתק טקסט
```
1. לחץ בתוך ה-Console
2. Ctrl+A (בחר הכל)
3. Ctrl+C (העתק)
4. הדבק כאן בצ'אט
```

## 💡 טיפים

### אם אתה רואה הרבה הודעות:
- זה בסדר! שלח הכל
- אני אדע לסנן את מה שחשוב

### אם אתה רואה הודעות באדום:
- מצוין! זה בדיוק מה שאני צריך
- וודא שאתה מעתיק גם אותן

### אם אתה לא רואה כלום:
- אולי ה-Console לא פתוח
- נסה שוב: F12 → Console → נקה → לחץ מיקרופון
