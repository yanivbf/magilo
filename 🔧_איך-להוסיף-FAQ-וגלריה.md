# 🔧 איך להוסיף FAQ וגלריה לדף

## הבעיה
אתה לא רואה את המקטעים החדשים (FAQ, גלריה) כי הדף שלך לא מכיל מידע בשדות האלה.

## הפתרון - 2 דרכים

### דרך 1: דרך ממשק הניהול (מומלץ) ✅

1. **היכנס לממשק הניהול של הדף:**
   ```
   http://localhost:3000/manage/[pageId]
   ```

2. **לחץ על הטאב "תוכן מקטעים"** (📝)

3. **הוסף שאלות ותשובות:**
   - לחץ על "+ הוסף שאלה"
   - מלא את השאלה והתשובה
   - חזור על זה כמה פעמים

4. **הוסף תמונות לגלריה:**
   - לחץ על "+ הוסף תמונה"
   - הכנס קישור לתמונה (למשל מ-Unsplash)
   - דוגמאות לקישורים:
     ```
     https://images.unsplash.com/photo-1505740420928-5e560c06d30e?w=800
     https://images.unsplash.com/photo-1523275335684-37898b6baf30?w=800
     https://images.unsplash.com/photo-1572635196237-14b3f281503f?w=800
     ```

5. **הוסף המלצות לקוחות:**
   - לחץ על "+ הוסף המלצה"
   - מלא שם, טקסט, ודירוג
   - חזור על זה כמה פעמים

6. **שמור שינויים:**
   - לחץ על "💾 שמור שינויים"
   - חכה להודעת הצלחה

7. **רענן את הדף:**
   - לך לדף הציבורי שלך
   - רענן (F5)
   - תראה את המקטעים החדשים!

---

### דרך 2: ישירות דרך Strapi Admin (מתקדם)

1. **היכנס ל-Strapi Admin:**
   ```
   http://localhost:1337/admin
   ```

2. **לך ל-Content Manager → Pages**

3. **בחר את הדף שלך**

4. **גלול למטה למצוא את השדות:**
   - `faq` - הוסף JSON:
     ```json
     [
       {
         "question": "מה שעות הפעילות?",
         "answer": "אנחנו פתוחים כל יום בין 9:00-18:00"
       },
       {
         "question": "איפה אתם נמצאים?",
         "answer": "אנחנו נמצאים בתל אביב, רחוב הרצל 123"
       }
     ]
     ```

   - `gallery` - הוסף JSON:
     ```json
     [
       "https://images.unsplash.com/photo-1505740420928-5e560c06d30e?w=800",
       "https://images.unsplash.com/photo-1523275335684-37898b6baf30?w=800",
       "https://images.unsplash.com/photo-1572635196237-14b3f281503f?w=800"
     ]
     ```

   - `testimonials` - הוסף JSON:
     ```json
     [
       {
         "name": "יוסי כהן",
         "text": "שירות מעולה! ממליץ בחום",
         "rating": 5
       },
       {
         "name": "שרה לוי",
         "text": "חוויה נהדרת, אחזור בוודאות",
         "rating": 5
       }
     ]
     ```

5. **שמור את השינויים**

6. **רענן את הדף הציבורי**

---

## בדיקה מהירה

אחרי שהוספת מידע, בדוק:

1. **FAQ מופיע?**
   - גלול לסוף הדף
   - תראה מקטע "❓ שאלות ותשובות"
   - לחץ על שאלה - היא תיפתח

2. **גלריה מופיעה?**
   - תראה מקטע "🖼️ גלריית תמונות"
   - לחץ על תמונה - תיפתח ב-lightbox
   - חצים לניווט בין תמונות

3. **המלצות מופיעות?**
   - תראה מקטע "⭐ מה אומרים עלינו"
   - כרטיסים עם שמות וכוכבים

---

## אם עדיין לא רואה

1. **בדוק ב-Console:**
   - פתח Developer Tools (F12)
   - לך ל-Console
   - חפש שגיאות

2. **בדוק שהמידע נשמר:**
   - פתח Network tab
   - רענן את הדף
   - חפש את הקריאה ל-API
   - בדוק שהמידע חוזר

3. **נקה Cache:**
   - Ctrl + Shift + R (hard refresh)
   - או Ctrl + F5
