# 🎯 מה לעשות עכשיו - מדריך מהיר

## ✅ מה תוקן?

הדפים עכשיו מוצגים עם קומפוננטות Svelte אינטראקטיביות במקום HTML סטטי.

---

## 🚀 איך לבדוק?

### שלב 1: הפעל את השרתים

```bash
# Terminal 1 - Strapi
cd strapi-backend
npm run develop

# Terminal 2 - SvelteKit
cd new-app
npm run dev
```

### שלב 2: צור דף חנות חדש

1. פתח: `http://localhost:5173/page-creator`
2. בחר: **חנות מקוונת**
3. מלא:
   - שם העסק: "החנות שלי"
   - תיאור: "המוצרים הכי טובים"
   - טלפון: "050-1234567"
4. הוסף **לפחות 3 מוצרים** עם:
   - שם מוצר
   - מחיר
   - תמונה (או השאר את ברירת המחדל)
5. לחץ **"צור דף"**

### שלב 3: צפה בדף

1. לך ל-`/dashboard`
2. לחץ **"👁️ צפה בדף"** על הדף שיצרת
3. **בדוק שאתה רואה:**
   - ✅ תמונת כותרת עם גרדיאנט סגול-ורוד
   - ✅ המוצרים בכרטיסיות יפות
   - ✅ כפתור "הוסף לסל" על כל מוצר
   - ✅ פרטי קשר בתחתית
   - ✅ סרגל עריכה למעלה (אם אתה הבעלים)
   - ✅ בוט למטה מימין

## ❌ אם זה לא עובד

### בעיה 1: לא רואה מוצרים

**פתרון:**
1. בדוק ש**הוספת מוצרים** בעת יצירת הדף
2. או לך ל-`/manage/[slug]` והוסף מוצרים
3. או ערוך את הדף ב-`/page-creator` והוסף מוצרים

### בעיה 2: הדף נראה כמו HTML פשוט

**פתרון:**
1. **רענן את הדף** (Ctrl+R או F5)
2. **נקה cache** (Ctrl+Shift+R)
3. **צור דף חדש** - דפים ישנים עשויים להיות עם HTML ישן

### בעיה 3: שגיאה 404

**פתרון:**
1. בדוק שהשרתים רצים (Strapi + SvelteKit)
2. בדוק שהדף נשמר ב-Strapi
3. נסה לגשת דרך `/pages/[slug]` במקום `/view/[slug]`
