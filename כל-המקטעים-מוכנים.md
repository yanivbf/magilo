## 🎉 מה נוצר

יצרתי 3 מקטעים חדשים שחסרו:

### 1. **🛠️ מקטע שירותים** (ServicesSection)
- כרטיסי שירותים עם אייקונים
- תמיכה באימוג'י או תמונה

- חבילות תמחור
- רשימת תכונות לכל חבילה
- אפשרות להדגיש חבילה מומלצת
- הוספה/מחיקה של חבילות ותכונות

- כרטיסי חברי צוות

- שם, תפקיד, תיאור
- הוספה/מחיקה של חברי צוות

## 🧪 איך להוסיף מקטע לדף

### דרך Strapi:
1. פתח את הדף
2. הוסף מקטע חדש
3. בחר את הסוג (services/pricing/team)
4. מלא את הנתונים
5. שמור

### דוגמה למקטע שירותים:
```json
{
  "type": "services",
  "enabled": true,
  "data": {
    "title": "🛠️ השירותים שלנו",
    "subtitle": "מגוון שירותים איכותיים",
    "services": [
      {
        "icon": "🔧",
        "title": "תיקונים",
        "description": "תיקון מקצועי",
        "price": "₪150"
      }
    ]
  }
}
```

### דוגמה למקטע מחירון:
```json
{
  "type": "pricing",
  "enabled": true,
  "data": {
    "title": "💰 מחירון",
    "subtitle": "בחר את החבילה המתאימה",
    "plans": [
      {
        "name": "בסיסי",

  }
}
```
