# 🎯 דשבורד משופר - כמו המערכת הישנה

## מה שופר?

### 1. **סטטוס Premium** ⭐
עכשיו הדשבורד מציג את סטטוס המנוי של המשתמש:

#### משתמש רגיל (Free):
```
┌─────────────────────────────┐
│ 👤 שם המשתמש                │
│ email@example.com           │
│ [משתמש רגיל]                │
└─────────────────────────────┘
```

#### משתמש Premium:
```
┌─────────────────────────────┐
│ 👤 שם המשתמש                │
│ email@example.com           │
│ [⭐ Premium] [פעיל עד ...]  │
└─────────────────────────────┘
```

### 2. **באנר שדרוג ל-Premium** 🎨
אם המשתמש לא Premium, מופיע באנר מושך עין:

```
╔═══════════════════════════════════════════════════╗
║  ⭐ שדרג ל-Premium                                ║
║  קבל גישה לכל התכונות המתקדמות:                  ║
║  ✓ הסרת מיתוג AutoPage                           ║
║  ✓ דומיין מותאם אישית                            ║
║  ✓ אנליטיקס מתקדם                                ║
║  ✓ תמיכה עדיפות 24/7                             ║
║                                                   ║
║                      [הפעל מנוי עכשיו]           ║
╚═══════════════════════════════════════════════════╝
```

```
┌─────────────────────────────┐
│ 🛍️ חנות מקוונת              │
│ ┌─────────────────────────┐ │
│ │ ⭐ Premium | פעיל       │ │  ← אם יש מנוי

│ או:                          │
│ ┌─────────────────────────┐ │
│ │ חינם | שדרג ל-Premium→ │ │  ← אם אין מנוי
```


#### כללי:

## 🔄 תהליך עבודה

### 1. כניסה לדשבורד
```
1. משתמש מתחבר
2. מערכת טוענת את הדפים שלו
3. מערכת בודקת סטטוס Premium
4. מציגה את הדשבורד המלא
```

### 2. צפייה בדף
```
1. לחיצה על "צפה"
2. פתיחת הדף בטאב חדש
3. הדף נפתח במצב צפייה ציבורי
```

### 3. עריכת דף
```
1. לחיצה על "ערוך"
2. מעבר ל-Page Creator במצב עריכה
3. טעינת כל הנתונים של הדף
4. אפשרות לערוך ולשמור
```

### 4. ממשק ניהול
```
1. לחיצה על "ממשק ניהול [סוג]"
2. מעבר לממשק הניהול המתאים
3. גישה לכל הכלים הרלוונטיים:
   - חנות: מוצרים, הזמנות, מלאי
   - אירוע: אורחים, RSVP, שולחנות
   - תורים: לקוחות, תורים, לוח שנה
```

### 5. שדרוג ל-Premium
```
1. לחיצה על "הפעל מנוי עכשיו"
2. פתיחת חלון תשלום (בעתיד)
3. הפעלת המנוי
4. עדכון הדשבורד עם סטטוס Premium
```

## 🚀 איך להשתמש?

### 1. פתח את הדשבורד
```
http://localhost:5173/dashboard
```

### 2. תראה את כל הדפים שלך
כל דף מוצג בכרטיס עם:
- תמונת תצוגה מקדימה
- שם הדף
- סוג הדף
- סטטוס מנוי
- כפתורי פעולה

### 3. לחץ על "ממשק ניהול"
- **חנות**: מעבר לניהול מוצרים והזמנות
- **אירוע**: מעבר לניהול אורחים ו-RSVP
- **תורים**: מעבר לניהול תורים ולקוחות

### 4. שדרג ל-Premium
לחץ על "הפעל מנוי עכשיו" כדי לקבל:
- הסרת מיתוג
- דומיין מותאם
- אנליטיקס
- תמיכה VIP

## 📝 הערות חשובות

1. **סטטוס Premium נשמר ב-Strapi** - בטבלת Users
2. **כל דף יכול להיות Premium בנפרד** - או כל המשתמש
3. **הממשק ניהול משתנה לפי סוג הדף** - אוטומטי
4. **הכפתורים עם אייקונים SVG** - מהירים ויפים
5. **הכל responsive** - עובד מצוין במובייל
