# ✅ כפתורים פשוטים במקום סרגל עריכה

## מה שונה?

### לפני:
```
┌─────────────────────────────────────────────┐
│ [העלאת תמונה] [מצב עריכה מלא] [לדף הבית] │ ← סרגל עם 3 כפתורים
└─────────────────────────────────────────────┘
```

### אחרי:
```
┌──────────────────┐
│ 📸 העלה תמונה    │ ← כפתור סגול
├──────────────────┤
│ 💾 שמור לאזור שלי │ ← כפתור ורוד
└──────────────────┘
```

## ✅ מה תוקן?

### 1. הסרת הסרגל העליון
- ❌ הוסר הסרגל עם 3 הכפתורים
- ✅ נשארו רק 2 כפתורים חיוניים

### 2. כפתור "העלה תמונה"
- 📸 **מיקום**: שמאל למעלה (קבוע)
- 🎨 **עיצוב**: סגול מדורג
- ⚡ **פעולה**: פותח בחירת תמונה → מעלה → מעדכן את תמונת הרקע

### 3. כפתור "שמור לאזור שלי"
- 💾 **מיקום**: מתחת לכפתור העלאת תמונה
- 🎨 **עיצוב**: ורוד מדורג
- ⚡ **פעולה**: שומר את הדף לדשבורד → מעבר לדשבורד

## 🎯 איך זה עובד?

### כשאתה הבעלים של הדף:
1. ✅ תראה 2 כפתורים בצד שמאל למעלה
2. ✅ תוכל ללחוץ על טקסט/תמונות לערוך אותם ישירות
3. ✅ כל שינוי נשמר אוטומטית
4. ✅ לחיצה על "שמור לאזור שלי" → מעבר לדשבורד

### כשאתה לא הבעלים:
1. ❌ לא תראה את הכפתורים
2. ❌ לא תוכל לערוך
3. ✅ תראה את הדף במצב צפייה רגיל
