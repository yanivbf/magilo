# 🔧 התחברות זמנית - פתרון מיידי!

## הבעיה:
- Google OAuth לא עובד (צריך הגדרות ב-Google Console)
- Email/Password לא עובד (Strapi משתמש במערכת אחרת)

## הפתרון המיידי:

**נשתמש ב-userId הקיים שלך!**

אתה כבר מחובר עם: `ef0bfa04-0d2a-48b0-8c0e-67c8b4ed0241`

## איך להתחבר עכשיו:

### אפשרות 1: דף התחברות מיוחד (הכי פשוט!)

1. **פתח את הדף הזה:**
   ```
   http://localhost:5173/set-user
   ```

2. **תראה כפתור "התחבר כמשתמש קיים"**

3. **לחץ עליו**

4. **זהו! אתה מחובר!**

---

### אפשרות 2: Console (אם אפשרות 1 לא עובדת)

1. **פתח את האתר:**
   ```
   http://localhost:5173/login
   ```

2. **פתח Console (F12)**

3. **הדבק את הקוד הזה:**
   ```javascript
   document.cookie = 'userId=ef0bfa04-0d2a-48b0-8c0e-67c8b4ed0241; path=/; max-age=2592000';
   localStorage.setItem('currentUser', JSON.stringify({
     id: 'ef0bfa04-0d2a-48b0-8c0e-67c8b4ed0241',
     userId: 'ef0bfa04-0d2a-48b0-8c0e-67c8b4ed0241',
     email: 'user@test.com',
     name: 'משתמש בדיקה'
   }));
   window.location.href = '/dashboard';
   ```

4. **לחץ Enter**

5. **זהו! אתה מחובר!**

---

## מה יקרה?

1. ✅ Cookie נשמר עם ה-userId שלך
2. ✅ המערכת מזהה אותך
3. ✅ אתה מועבר לדשבורד
4. ✅ תוכל ליצור דפים חדשים
5. ✅ אין שגיאה 500!
