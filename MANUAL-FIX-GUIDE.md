## 🔧 איך לתקן:

### אופציה 1: עריכה ידנית
1. פתח את `public/index.html` בעורך טקסט (VS Code, Notepad++, וכו')
2. עבור לכל שורה מהרשימה למעלה
3. החלף את הטקסט הגיבריש בטקסט הנכון
4. **חשוב!** שמור את הקובץ ב-UTF-8 encoding

### אופציה 2: שימוש בעורך עם Find & Replace
1. פתח את הקובץ ב-VS Code
2. לחץ Ctrl+H (Find & Replace)
3. העתק את הטקסט הגיבריש מעמודה "מה כתוב עכשיו"
4. הדבק בשדה "Find"
5. העתק את הטקסט הנכון מעמודה "צריך להיות"
6. הדבק בשדה "Replace"
7. לחץ "Replace All"
