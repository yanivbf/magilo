# ✅ רמקול בצ'אט מוכן!

## מה הוספתי?

### 🔊 כפתור רמקול בכל הודעה של הבוט

**בשני הצ'אטים:**
1. **ChatBotBubble** - הצ'אט הכללי
2. **StavBot** - הצ'אט החכם במרקטפלייס

### איך זה עובד?

**כל הודעה של הבוט מקבלת כפתור רמקול קטן מתחת להודעה:**

- 🔊 **לחיצה ראשונה** - מקריא את ההודעה בקול
- ⏸️ **לחיצה שנייה** - עוצר את ההקראה

### טכנולוגיה

**שתי שיטות TTS (Text-to-Speech):**

1. **Google Cloud TTS** (אם מוגדר API Key)
   - קול נשי עברי איכותי
   - דורש הגדרת `GOOGLE_TTS_API_KEY` ב-.env

2. **Browser TTS** (גיבוי אוטומטי)
   - משתמש ב-Web Speech API של הדפדפן
   - עובד בלי הגדרות נוספות
   - קול עברי בסיסי

### איפה לראות?

1. פתח את האתר
2. לחץ על כפתור הצ'אט (הבועה הסגולה)
3. שלח הודעה
4. תראה כפתור רמקול קטן מתחת לתשובה של הבוט
5. לחץ עליו כדי לשמוע את ההודעה

## הערות

- הרמקול מנקה אוטומטית סימנים מיוחדים (אימוג'י, קישורים)
- אם יש שגיאה ב-Google TTS, עובר אוטומטית לדפדפן
- ניתן לעצור הקראה באמצע על ידי לחיצה שוב
- הקול המדויק של סתיו: `he-IL-Wavenet-A` עם `speakingRate: 1.1`
