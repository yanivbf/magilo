# 📋 הסבר: מערכת המקטעים החדשה

## הבעיה שראית
הדף הציג HTML גולמי במקום רכיבים מעוצבים.

## למה זה קרה?
המערכת הישנה שמרה מקטעים כ-HTML גולמי ב-Strapi.
המערכת החדשה משתמשת ברכיבי Svelte מעוצבים.

## המערכת החדשה

### רכיבי Section מובנים (במקום HTML גולמי):

1. **GallerySection** - גלריית תמונות מעוצבת
   - תמונות עם אפקטים
   - Lightbox למסך מלא
   - אנימציות

2. **FAQSection** - שאלות ותשובות
   - עיצוב מודרני
   - אנימציות
   - אייקונים

3. **TestimonialsSection** - המלצות לקוחות
   - כרטיסים מעוצבים
   - כוכבים
   - אנימציות

4. **AboutSection** - אודות
   - טקסט מעוצב
   - תמונות
   - תכונות

### איך להשתמש?

במקום להוסיף מקטעים ידנית, המערכת מזהה אוטומטית:

```javascript
// אם יש gallery - מציג GallerySection
{#if gallery && gallery.length > 0}
  <GallerySection data={{ images: gallery }} />
{/if}

// אם יש FAQ - מציג FAQSection
{#if page.faq && page.faq.length > 0}
  <FAQSection data={{ items: page.faq }} />
{/if}

// אם יש testimonials - מציג TestimonialsSection
{#if page.testimonials && page.testimonials.length > 0}
  <TestimonialsSection data={{ items: page.testimonials }} />
{/if}

// אם יש aboutText - מציג AboutSection
{#if page.aboutText}
  <AboutSection data={{ content: page.aboutText }} />
{/if}
```

## מה השתנה?

### לפני (מערכת ישנה):
- מקטעים נשמרו כ-HTML גולמי
- לא היה עיצוב אחיד
- קשה לעדכן
- נראה לא מקצועי

### אחרי (מערכת חדשה):
- רכיבי Svelte מעוצבים
- עיצוב אחיד ומקצועי
- קל לעדכן
- אנימציות ואפקטים

## איך להוסיף תוכן?

### דרך דף הניהול:
1. לך ל-`/manage/{pageId}`
2. הוסף תוכן לשדות המתאימים:
   - Gallery - העלה תמונות
   - FAQ - הוסף שאלות ותשובות
   - Testimonials - הוסף המלצות
   - About - כתוב טקסט אודות

### דרך Strapi:
1. פתח את הדף ב-Strapi
2. ערוך את השדות:
   - `gallery` - מערך של URLs לתמונות
   - `faq` - מערך של {question, answer}
   - `testimonials` - מערך של {text, author, rating}
   - `aboutText` - טקסט חופשי

## יתרונות המערכת החדשה

✅ אנימציות ואפקטים
