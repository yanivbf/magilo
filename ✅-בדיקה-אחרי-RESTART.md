# ✅ בדיקה מהירה אחרי RESTART

## 1️⃣ בדיקה בסיסית

### פתח דף
```
http://localhost:5173/view/[slug]
```

### בדוק שהדף נטען
- [ ] הדף נטען ללא שגיאות
- [ ] אין שגיאות אדומות ב-Console
- [ ] כל המקטעים מוצגים

## 2️⃣ בדיקת עריכה - Hero

### כותרת
1. לחץ על הכותרת הראשית
2. ערוך את הטקסט
3. לחץ מחוץ לשדה
4. **בדוק:** האם הופיעה הודעה ירוקה "✅ נשמר בהצלחה"?

### תיאור
1. לחץ על התיאור
2. ערוך את הטקסט
3. לחץ מחוץ לשדה
4. **בדוק:** האם נשמר?

## 3️⃣ בדיקת עריכה - Gallery

### תמונה
1. לחץ על תמונה בגלריה
2. **בדוק:** האם מופיע overlay עם אייקון מצלמה?
3. בחר תמונה חדשה
4. **בדוק:** האם התמונה מוחלפת?

### כותרת
1. לחץ על כותרת הגלריה
2. ערוך
3. **בדוק:** האם נשמר?

## 4️⃣ בדיקת עריכה - Products

### שם מוצר
1. לחץ על שם מוצר
2. ערוך
3. **בדוק:** האם נשמר?

### מחיר
1. לחץ על המחיר
2. שנה את המספר
3. **בדוק:** האם נשמר?

### תמונת מוצר
1. לחץ על תמונת מוצר
2. **בדוק:** האם מופיע overlay?
3. בחר תמונה
4. **בדוק:** האם מוחלפת?

## 5️⃣ בדיקת כפתורים

### כפתור ➕
1. גלול למקטע Products
2. לחץ על כפתור ➕
3. **בדוק:** האם נוסף מוצר חדש?

### כפתור 🗑️
1. לחץ על כפתור 🗑️ ליד מוצר
2. אשר את המחיקה
3. **בדוק:** האם המוצר נמחק?

## 6️⃣ בדיקת Console

### פתח Developer Tools
```
F12 או Ctrl+Shift+I
```

### בדוק Console
- [ ] אין שגיאות אדומות
- [ ] רואה הודעות ירוקות (✅)
- [ ] רואה לוגים של שמירה (💾)

### בדוק Network
1. לך ל-Network tab
2. ערוך משהו
3. **בדוק:** האם הבקשה הצליחה (200 OK)?

## 7️⃣ בדיקת Responsive

### טאבלט
1. פתח Developer Tools (F12)
2. לחץ על אייקון המכשירים (Toggle device toolbar)
3. בחר iPad
4. **בדוק:** האם הדף נראה טוב?

### מובייל
1. בחר iPhone
2. **בדוק:** האם הדף נראה טוב?
3. **בדוק:** האם כל הכפתורים נגישים?

**פתרון:** בדוק Console לשגיאות

### תמונות לא מוצגות
**פתרון:** בדוק את ה-URL ב-Network tab
