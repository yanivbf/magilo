## מתי הבאנר מופיע?

הבאנר מופיע **רק** כאשר:

1. ✅ המשתמש הוא **בעלים של הדף** (`data.isOwner === true`)
2. ✅ אין לדף **מנוי פעיל** (`data.page.subscriptionStatus !== 'active'`)

### דוגמאות:

#### תרחיש 1: בעלים ללא מנוי
```
משתמש: ofer.dadush (בעלים)
מנוי: לא פעיל
→ הבאנר מופיע ✅
```

#### תרחיש 2: בעלים עם מנוי
```
משתמש: ofer.dadush (בעלים)
מנוי: פעיל
→ הבאנר לא מופיע ❌
```

#### תרחיש 3: מבקר (לא בעלים)
```
משתמש: אורח
מנוי: לא רלוונטי
→ הבאנר לא מופיע ❌
```

## איך לבדוק?

### 1. התחבר כבעלים
```
1. לך ל-http://localhost:5173/login
2. התחבר עם המשתמש שיצר את הדף
```

### 2. פתח את הדף
```
1. לך ל-http://localhost:5173/view/[slug]
2. החלף [slug] בשם הדף שלך
```

### 3. בדוק את הבאנר
```
✅ הבאנר אמור להופיע אחרי ה-Hero
✅ הכוכבים אמורים לפעום
✅ הכפתור אמור להגיב ל-hover

```

### 4. בדוק עם מנוי פעיל
```
1. שנה את subscriptionStatus ל-'active' ב-Strapi
2. רענן את הדף
3. הבאנר לא אמור להופיע ✅
```

### 5. בדוק כאורח
```
1. התנתק
2. פתח את הדף
3. הבאנר לא אמור להופיע ✅
```
