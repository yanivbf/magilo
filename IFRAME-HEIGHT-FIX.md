## ✅ **התיקון:**

### **לפני:**
```javascript
<iframe ... style="width: 100%; height: 2400px; ..." />
//                           ^^^^^^^^
//                           גובה קבוע ענקי!
```

### **אחרי:**
```javascript
<iframe ... style="width: 100%; height: 100%; ..." />
//                           ^^^^^^^^^
//                           גובה מתאים!
```

---

## 🔧 **מה שונה:**

### **1. גובה iframe: 100% במקום 2400px**
```javascript
// לפני:
style="width: 100%; height: 2400px; ..."

// אחרי:
style="width: 100%; height: 100%; ..."
```

### **2. Container overflow: hidden במקום auto**
```javascript
// לפני:
<div style="... overflow: auto; ...">

// אחרי:
<div style="... overflow: hidden; ...">
```

**למה?**
- כשה-iframe ב-`height: 100%`, הוא ממלא את כל ה-container
- הגלילה קורה **בתוך ה-iframe עצמו**, לא בתוך ה-container
- אין צורך ב-scrollbar כפול
