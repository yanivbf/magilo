# 🎨 מדריך מהיר: סגנונות חדשים

## 🎯 מה יש לנו?

### 3 סגנונות חדשים:

#### 1. 🎭 קומיקס (Comic Book)
**מתי להשתמש:** לעסקים צעירים, אירועים כיפיים, מוצרים יצירתיים

**מה תקבל:**
- גבולות שחורים עבים (4px)
- צללים "בלוקיים" קשיחים
- צבעים רוויים (צהוב, אדום, כחול)
- טקסט מודגש ב-UPPERCASE

**דוגמה:**
```
┌─────────────────────┐
│  ⭐⭐⭐⭐⭐         │
│                     │
│  "שירות מעולה!"    │
│                     │
│  יוסי כהן          │
│  לקוח מרוצה        │
└─────────────────────┘
   ▼▼▼▼▼▼▼▼▼▼▼▼▼▼▼
   צל שחור קשיח
```

---

#### 2. 📜 רטרו וינטג' (Retro Vintage)
**מתי להשתמש:** לעסקים מסורתיים, מסעדות, חנויות עתיקות

**מה תקבל:**
- גוונים חומים-ספיה
- טקסטורת רעש (נייר ישן)
- פונטים Serif (מכונת כתיבה)
- גבולות כפולים דקורטיביים

**דוגמה:**
```
╔═══════════════════╗
║                   ║
║  ⭐⭐⭐⭐⭐      ║
║                   ║
║  "שירות מעולה!"  ║
║                   ║
║  יוסי כהן        ║
║  לקוח מרוצה      ║
║                   ║
╚═══════════════════╝
  (טקסטורת נייר)
```

---

#### 3. 🌈 הולוגרפי תלת מימד (Holographic 3D)
**מתי להשתמש:** לעסקים טכנולוגיים, סטארט-אפים, מוצרים עתידניים

**מה תקבל:**
- אפקטי 3D (rotateY, rotateX)
- זוהר ניאון סביב הכרטיסים
- שקיפות עם blur (זכוכית)
- אנימציית ריחוף איטית

**דוגמה:**
```
    ╱╲╱╲╱╲╱╲╱╲
   ╱  זוהר   ╲
  ╱ ┌─────────┐ ╲
 ╱  │ ⭐⭐⭐⭐⭐│  ╲
╱   │         │   ╲
    │ "מעולה!" │
    │         │
    │ יוסי כהן│
    └─────────┘
   (מרחף באוויר)
```

## 🚀 איך להשתמש?

### צעד 1: פתח את SmartPageCreator
```
http://localhost:5177/page-creator
```

### צעד 2: בחר סוג דף
- אמן / מוזיקאי 🎤
- מסעדה 🍽️
- חנות מקוונת 🛍️
- או כל סוג אחר...

### צעד 3: מלא פרטים בסיסיים
- שם העסק
- תיאור
- טלפון
- וכו'

### צעד 4: בחר סגנון עיצוב
גלול למטה עד שתראה את הסגנונות:

```
┌─────────┐ ┌─────────┐ ┌─────────┐
│ קומיקס  │ │  רטרו   │ │ תלת מימד│
│  🎭     │ │   📜    │ │   🌈    │
└─────────┘ └─────────┘ └─────────┘
```

**לחץ על הסגנון שאתה רוצה!**

### צעד 5: סמן מקטעים
סמן את המקטעים שאתה רוצה:
- ✅ המלצות (Testimonials)
- ✅ שאלות ותשובות (FAQ)
- ✅ צוות (Team)

### צעד 6: צור דף
לחץ על "צור דף" והמתן...

### צעד 7: תהנה!
המקטעים יופיעו עם הסגנון שבחרת! 🎉
