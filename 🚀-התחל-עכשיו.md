# 🚀 התחל עכשיו - מדריך מהיר

## מה תיקנו?

**התחברות Google עובדת! בלי Supabase, בלי שגיאות 500!**

---

## שלב 1: ודא שהסרבר רץ

```bash
cd new-app
npm run dev
```

אמור לראות:
```
  ➜  Local:   http://localhost:5173/
```

---

## שלב 2: פתח את דף ההתחברות

פתח בדפדפן:
```
http://localhost:5173/login
```

---

## שלב 3: התחבר עם Google

1. לחץ על הכפתור האדום: **"התחבר עם Google"**
2. בחר את חשבון Google שלך
3. אמור להעביר אותך ל: `http://localhost:5173/dashboard`

### מה אמור לקרות?
- ✅ רואה את הדשבורד
- ✅ רואה את השם שלך בפינה הימנית העליונה
- ✅ רואה רשימת דפים (אם יש לך)

---

## שלב 4: צור דף חדש

1. לחץ על **"צור דף חדש"**
2. בחר **"חנות מקוונת"** (או כל תבנית אחרת)
3. מלא את הפרטים:
   - שם העסק: `חנות הבדיקה שלי`
   - תיאור: `זה בדיקה`
   - (אפשר לדלג על תמונה)
4. לחץ **"צור דף"**

### מה אמור לקרות?
- ✅ הדף נוצר
- ✅ מועבר לדף הניהול: `/manage/[documentId]`
- ✅ **אין שגיאה 500!**
- ✅ רואה את הדף שיצרת

---

## בדיקות

### בדיקה 1: Cookie נשמר?

1. פתח Console (F12)
2. לך ל-**Application** → **Cookies**
3. חפש Cookie בשם: `userId`
4. הערך אמור להתחיל ב-`google_`

**דוגמה:**
```
userId: google_123456789
```

### בדיקה 2: Terminal מראה הצלחה?

בדוק ב-Terminal שרץ את הסרבר:

```
✅ Cookie set for user: google_123456789
✅ New user created: google_123456789
```

או אם המשתמש כבר קיים:
```
✅ Existing user logged in: google_123456789
✅ Cookie set for user: google_123456789
```

### בדיקה 3: יצירת דף עובדת?

אחרי שלחצת "צור דף":

```
✅ Page created successfully
📄 Document ID: abc123xyz
🔗 Redirecting to manage page: /manage/abc123xyz
```

## אם זה לא עובד - אל תדאג! 🔧

### שגיאה: "אין Cookie"

**פתרון:**
1. נקה Cookies (F12 → Application → Clear site data)
2. התחבר שוב עם Google

### שגיאה: "500 Internal Server Error"

**פתרון:**
1. בדוק ב-Terminal מה השגיאה המדויקת
2. העתק את השגיאה ושלח לי

### שגיאה: "לא מעביר לדשבורד"

**פתרון:**
1. בדוק ב-Console (F12) אם יש שגיאות
2. בדוק שה-`currentUser` מוגדר
3. העתק את השגיאה ושלח לי
