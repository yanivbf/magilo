# 🛒 הסבר: איפה לערוך מוצרים בגלריה

## ❌ הבעיה שהיתה
ניסית לערוך מוצרים בדף הצפייה (`/view/[slug]`) - התמונות והטקסט לא נשמרו.

## ✅ הפתרון הנכון

### המוצרים נערכים רק בדף הניהול!

**דף הצפייה** (`/view/[slug]`) = לצפייה וקניות בלבד
- הלקוחות רואים את המוצרים
- יכולים להוסיף לעגלה ולהזמין
- **אי אפשר לערוך מוצרים שם** (גם אם אתה הבעלים)

**דף הניהול** (`/manage/[pageId]`) = לעריכת מוצרים
- יש טאב "מוצרים" 🛍️
- שם אתה יכול:
  - ✅ להוסיף מוצר חדש
  - ✅ לערוך שם, מחיר, תיאור
  - ✅ להעלות תמונה
  - ✅ למחוק מוצר
  - ✅ להפעיל/להשבית מוצר

## 📋 איך לערוך מוצרים (צעד אחר צעד)

### 1️⃣ היכנס לדף הניהול
```
מהדשבורד → לחץ על "נהל" ליד הדף שלך
או
גש ישירות ל: /manage/[pageId]
```

### 2️⃣ לחץ על טאב "מוצרים" 🛍️
בחלק העליון של דף הניהול יש טאבים:
- תוכן
- שירותים
- **מוצרים** ← לחץ כאן!
- מקטעים

### 3️⃣ נהל את המוצרים
- **הוסף מוצר חדש**: לחץ "+ הוסף מוצר"
- **ערוך מוצר קיים**: לחץ "ערוך" על המוצר
- **העלה תמונה**: בחר קובץ תמונה
- **שמור**: לחץ "שמור"

### 4️⃣ צפה בתוצאה
אחרי שמירה, גש לדף הצפייה (`/view/[slug]`) וראה את המוצרים המעודכנים!

## 🔧 מה תוקן בקוד

### בדף הצפייה (view):
```svelte
<ProductsGallerySection 
    data={{ ...section.data, products: data.page.products }} 
    editable={false}  ← לא ניתן לעריכה!
/>
```

### בדף הניהול (manage):
```svelte
{:else if activeTab === 'products'}
    <ProductManager 
        {pageId} 
        products={pageData?.products || []} 
    />
```

## 💡 למה זה עובד ככה?

1. **הפרדה נכונה**: 
   - דף צפייה = לקוחות
   - דף ניהול = בעל העסק

2. **שמירה נכונה**:
   - ProductManager שומר ישירות לטבלת `products` ב-Strapi
   - כל מוצר נשמר בנפרד עם ID משלו
   - התמונות נשמרות ב-Strapi Media Library

3. **טעינה נכונה**:
   - דף הצפייה טוען את המוצרים מ-`data.page.products`
   - זה מגיע מהשרת (Strapi)
   - תמיד מעודכן

## 🎯 סיכום

| מה אתה רוצה לעשות | איפה לעשות את זה |
|-------------------|------------------|
| לערוך מוצרים | `/manage/[pageId]` → טאב "מוצרים" |
| לראות את המוצרים | `/view/[slug]` |
| לקנות מוצרים | `/view/[slug]` → הוסף לעגלה |

כל השינויים במוצרים (תמונות, טקסט, מחיר) נשמרים ונטענים נכון.

פשוט תשתמש בדף הניהול לעריכה! 🎉
