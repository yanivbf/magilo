# 📋 תוכנית השלמת החנות - פשוט וברור

## 🎯 המטרה:
לגרום לחנות החדשה לעבוד **בדיוק** כמו הישנה.

## ✅ מה כבר עשינו:

1. ✅ העתקנו את התבנית הישנה (`store-legacy.html`)
2. ✅ הוספנו placeholders למילוי נתונים
3. ✅ יצרנו webhook ל-n8n (`/api/n8n-webhook`)
4. ✅ הוספנו יצירת HTML למוצרים

## ❌ מה עדיין חסר:

### 1. תוכן במקטעים (אודות, גלריה, וכו')
**הבעיה:** המקטעים ריקים - אין טקסט.

**הפתרון:**
- **אופציה A:** לקרוא לבוט n8n בזמן יצירת הדף (לוקח זמן)
- **אופציה B:** למלא תוכן ברירת מחדל פשוט (מהיר) ✅ **מומלץ**

### 2. בוט בועה בדף
**הבעיה:** הבוט של סתיו לא מופיע בפינה הימנית.

**הפתרון:**
- להוסיף את `PageBotBubble` או `StavBot` לתבנית
- לחבר אותו ל-webhook: `https://n8n-service-how4.onrender.com/webhook/jhfuhgufkhlkuho8erhf757754jhldkbsjkbkfhkfggyt`

### 3. בוט WhatsApp
**הבעיה:** בוט WhatsApp לא מופיע בפינה השמאלית.

**הפתרון:**
- להוסיף כפתור WhatsApp לתבנית (כבר קיים בתבנית הישנה!)

## 🚀 תוכנית הפעולה:

### שלב 1: מילוי תוכן ברירת מחדל (5 דקות)
```javascript
// ב-templateEngine.js
const defaultContent = {
  ABOUT_HTML: `

  `,
  GALLERY_HTML: `<!-- תמונות גלריה -->`,
  TESTIMONIALS_HTML: `<!-- המלצות לקוחות -->`,
  FAQ_HTML: `<!-- שאלות ותשובות -->`

```

- לבדוק שכפתור WhatsApp עובד (כבר קיים בתבנית!)
