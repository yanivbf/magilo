## 🎨 התאמה אישית

### אם התמונה קטנה מדי:
פתח `public/marketplace.html` וחפש:
```css
.stav-avatar {
    width: 120px;
    height: 120px;
}
```

שנה ל:
```css
.stav-avatar {
    width: 160px;  /* גדול יותר */
    height: 160px;
}
```

### אם התמונה לא עגולה:
אם אתה רוצה שהתמונה תהיה **לא עגולה** (מלבן), שנה:
```css
.stav-avatar-image {
    border-radius: 50%;  /* עגול */
}
```

ל:
```css
.stav-avatar-image {
    border-radius: 20px;  /* מעוגל רק בפינות */
}
```

או:
```css
.stav-avatar-image {
    border-radius: 0;  /* מלבן */
}
```
