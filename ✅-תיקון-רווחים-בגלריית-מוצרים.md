# ✅ תיקון רווחים בגלריית מוצרים

## הבעיה
היו רווחים **למעלה** ו**מימין** של התמונה בכרטיס המוצר.

## הסיבה
ה-`product-card` וה-`product-image-wrapper` היו עם padding/margin שיצרו רווחים לא רצויים.

## הפתרון

### 1. אפס padding ו-margin בכרטיס
```css
.product-card {
    flex-shrink: 0;
    width: 200px;
    border-radius: 16px;
    overflow: hidden;
    /* ✅ אפס padding ו-margin - בלי רווחים! */
    padding: 0 !important;
    margin: 0 !important;
}
```

### 2. אפס padding ו-margin ב-wrapper + border-radius מותאם
```css
.product-image-wrapper {
    position: relative;
    width: 200px;
    height: 200px;
    overflow: hidden;
    background: var(--color-bg-alt);
    /* ✅ אפס margin ו-padding - בלי רווחים! */
    margin: 0 !important;
    padding: 0 !important;
    /* ✅ border-radius רק בפינות העליונות */
    border-radius: 16px 16px 0 0;
}
```

## מה השתנה

1. **!important על padding ו-margin**: מבטיח שאין רווחים
2. **border-radius מותאם**: `16px 16px 0 0` - רק בפינות העליונות כדי להתאים לכרטיס
3. **overflow: hidden**: חותך כל מה שחורג

## תוצאה

✅ אין רווח מימין
✅ התמונה מכסה את כל החלק העליון של הכרטיס
✅ הפינות מעוגלות בצורה יפה
