# ✅ תיקון כותרות מרוכזות וירידות שורה

## מה תוקן?

### 1. כותרות מקטעים מרוכזות ✅
**בעיה**: הכותרות של המקטעים לא היו מרוכזות ולא בגודל מתאים.

**פתרון**: 
- עדכנתי את `new-app/src/app.css` להשתמש ב-`:global()` כדי לעקוף את הסגנונות המקומיים של כל קומפוננטה
- הוספתי `!important` לכל המאפיינים כדי להבטיח שהם יחולו
- כעת כל הכותרות:
  - **מרוכזות** באמצע הדף
  - **גודל מכובד**: 2.5rem (במובייל: 2rem)
  - **גרדיאנט סגול יפה** (כמו בשאר האתר)
  - **רספונסיביות מלאה** לכל גדלי מסך

### 2. ירידות שורה (Enter) בעריכת טקסט ✅
**בעיה**: כשלוחצים Enter בעריכת טקסט, השורה החדשה לא נשמרת.

**פתרון**:
- עדכנתי את `EditableText.svelte` לשמור את ה-HTML הפנימי במקום רק את הטקסט
- המערכת ממירה `<br>` ו-`<div>` לירידות שורה (`\n`) בשמירה
- בטעינה, ממירה בחזרה `\n` ל-`<br>` לתצוגה
- כעת ירידות שורה נשמרות ומוצגות כמו שצריך!

## איך לראות את השינויים?

### שלב 1: רענן את הדפדפן
לחץ **Ctrl+Shift+R** (או **Ctrl+F5**) כדי לרענן ולנקות את המטמון.

### שלב 2: בדוק כותרות
כנס לכל דף ותראה שכל הכותרות:
- מרוכזות באמצע
- בגודל גדול ומכובד
- עם גרדיאנט סגול יפה

### שלב 3: בדוק ירידות שורה
1. היכנס למצב עריכה
2. ערוך כותרת או טקסט
3. לחץ Enter לירידת שורה
4. שמור
5. רענן את הדף
6. תראה שהירידת שורה נשמרה!

## קבצים ששונו

1. **new-app/src/app.css** - סגנונות גלובליים לכותרות
2. **new-app/src/lib/components/editing/EditableText.svelte** - תמיכה בירידות שורה

## הערות

- אם הכותרות עדיין לא מרוכזות, נקה מטמון הדפדפן (Ctrl+Shift+Delete)
- ירידות שורה עובדות רק בטקסטים שנערכים דרך EditableText
- כל המקטעים משתמשים באותו סגנון כותרת אחיד
