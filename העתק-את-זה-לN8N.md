# 🚨 חשוב מאוד - תקן את סתיו!

## הבעיה שאתה רואה:
- סתיו מדברת עם ניקוד מוזר: "תַּשְׁלוּמִים. מְעַנְיֵן..."
- סתיו מוסיפה מספרים: "1. 2. 3. 4. 5. 6. 7..."
- התשובות לא הגיוניות

## הפתרון:

### 🔹 שלב 1: פתח את N8N
1. לך ל-N8N workflow של סתיו
2. מצא את הנוד שבו יש את ההנחיה (System Message)

### 🔹 שלב 2: העתק את ההנחיה החדשה
1. פתח את הקובץ: `N8N-SYSTEM-MESSAGE-FULL.txt`
2. **בחר הכל** (Ctrl+A)
3. **העתק** (Ctrl+C)

### 🔹 שלב 3: הדבק ב-N8N
1. חזור ל-N8N
2. **מחק את כל ההנחיה הישנה** בנוד
3. **הדבק את ההנחיה החדשה** (Ctrl+V)
4. **שמור את ה-Workflow**
5. **הפעל מחדש את ה-Workflow**

## ✅ מה יקרה אחרי זה:
- סתיו תדבר בעברית רגילה **ללא ניקוד**
- סתיו תוכל **לשוחח בחופשיות** (לא רק לחפש שירותים)
- לא יהיו יותר מספרים מוזרים
- התשובות יהיו הרבה יותר טבעיות וחכמות

## 📝 דוגמאות לשיחה חדשה:

**לפני:**
```
משתמש: "שלום"
סתיו: "שָׁלוֹם לְךָ גַּם. 1. 2. 3. מָה אַתָּה רוֹצֶה..."
```

**אחרי:**
```
משתמש: "שלום"
סתיו: "היי! מה נשמע?"

משתמש: "משעמם לי"
סתיו: "אני מבינה. רוצה שנשוחח על משהו?"

משתמש: "ספרי לי סיפור"
סתיו: "פעם היה אדם שחיפש סתם סיפור מבוט... אז הוא פגש אותי"
```
