## אופציה 2: תיקון ידני 🛠️

### שלב 1: נקה cookies
```javascript
// פתח Console בדפדפן (F12) והרץ:
document.cookie.split(";").forEach(function(c) { 
    document.cookie = c.replace(/^ +/, "").replace(/=.*/, "=;expires=" + new Date().toUTCString() + ";path=/"); 
});
```

פתח טרמינל חדש והרץ:
```bash
cd new-app
node -e "
fetch('http://localhost:5173/api/subscription/activate-user', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({userId: 'google_117656147186', months: 12})
}).then(r => r.json()).then(console.log)
"
```
