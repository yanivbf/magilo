## ✨ תכונות

### במצב עריכה:
- ✏️ עריכת טלפון, אימייל, כתובת
- 🔗 הוספת קישורים לרשתות חברתיות
- 💾 שמירה אוטומטית

### במצב צפייה:
- 📱 כרטיסי יצירת קשר צבעוניים
- 🌐 אייקונים של רשתות חברתיות עם כפתור "עקוב"
- 🎨 עיצוב מודרני עם אנימציות

## 🧪 איך להוסיף למערכת

### אופציה 1: דרך יוצר הדפים
כשיוצרים דף חדש, המקטע יתווסף אוטומטית.

### אופציה 2: הוספה ידנית ל-Strapi
הוסף מקטע חדש עם:
```json
{
  "type": "contact",
  "enabled": true,
  "data": {
    "title": "📞 צור קשר",
    "phone": "050-1234567",
    "email": "info@example.com",
    "address": "רחוב הדוגמה 123, תל אביב",
    "socialLinks": {
      "facebook": "https://facebook.com/yourpage",
      "instagram": "https://instagram.com/yourpage",
      "whatsapp": "972501234567",
      "youtube": "https://youtube.com/@yourchannel",
      "tiktok": "https://tiktok.com/@yourpage"
    }
  }
}
```

## 🎨 עיצוב

### כרטיסי יצירת קשר:
- רקע גרדיאנט סגול-ורוד
- אייקונים גדולים
- אנימציית hover

### רשתות חברתיות:
- כל רשת עם הצבע המקורי שלה
- אייקונים רשמיים
- כפתור "עקוב" אינטראקטיבי

## 💡 טיפים

1. **WhatsApp** - הזן מספר עם קידומת בינלאומית (972...)
2. **קישורים** - הזן URL מלא (https://...)
3. **עריכה** - פשוט לחץ על השדה וערוך
4. **שמירה** - אוטומטית כשיוצאים מהשדה

## 🔄 מה הלאה?

המקטע מוכן לשימוש! כדי שיופיע בדפים קיימים:
1. פתח את הדף ב-Strapi
2. הוסף מקטע חדש מסוג "contact"
3. מלא את הפרטים
4. שמור

הכל יעבוד אוטומטית! 🎉
